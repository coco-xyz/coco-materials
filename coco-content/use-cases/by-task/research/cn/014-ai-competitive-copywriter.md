# Use Case #014: AI Competitive Copywriter

**Role**: Marketing Strategist / Brand Manager / Competitive Intelligence Analyst | **Industry**: SaaS, Technology, E-commerce, B2B Services | **Task**: Competitive Analysis, Differentiation Messaging, Battle Card Creation, Win/Loss Analysis

---

## 社媒文案

最大的竞品发布了一个新功能，和我们产品60%重叠。一周内，销售团队丢了3个培育了好几个月的机会。客户都说同样的话："他们的宣传听起来和你们做的一样，但更便宜。"

痛苦的真相是：这不是真的。我们的产品在4个关键领域更强。但我们的消息传递没有说明这一点。网站文案在竞品发布后没更新。销售deck还在针对他们的旧产品定位。竞品作战卡已经过时6个月了。

那次警钟后我们部署了COCO的AI Competitive Copywriter：
- 每周监控竞品网站、新闻稿、G2评论和产品变更日志
- 竞品发布变更时自动更新我们的竞品作战卡
- 为每个渠道生成差异化聚焦的文案：网站、销售deck、邮件序列、广告文案
- 竞品发布新东西时，我们24小时内就有更新的定位文案——而不是3周
- 基于竞品的真实宣传创建异议处理脚本

6个月后：竞品对战胜率从34%提升到52%。因"竞品宣传"丢失的交易减少了61%。销售团队不再被打个措手不及。

## 详细介绍

**痛点：你的竞品营销信息永远过时**

在竞争激烈的市场中，消息传递不是静态的——它是一个不断变化的战场。竞品发布新功能、改变定价、更新网站文案、发布新案例研究、招聘新营销团队。每一次变化都可能改变潜客对竞争格局的感知。

大多数公司对竞争变化的响应是被动且缓慢的。竞品发布新功能——营销团队需要2-3周更新作战卡，4-6周更新网站，销售可能一个月后才听说。在这个滞后期间，交易被丢失了，因为销售在用过时的弹药战斗。

情报到行动的差距才是真正的问题。大多数组织有某种形式的竞品情报。但将情报转化为可操作的销售和营销文案——作战卡、异议处理、比较页面、邮件模板、广告文案——是一个手动的、耗时的过程，永远跟不上节奏。

**COCO如何解决**

COCO的AI Competitive Copywriter弥合竞品情报和面向收入的文案之间的差距。

1. **持续竞品监控**：COCO跟踪竞品活动：
   - 网站变更（定价页、功能页、首页消息）
   - 产品更新和变更日志
   - 新闻稿和博客文章
   - G2/Capterra评论（客户喜欢什么和讨厌什么）
   - 社交媒体公告
   - 招聘信息（揭示战略方向）
   - 生成每周竞品情报摘要

2. **动态作战卡生成**：竞品数据变化时，COCO自动更新：
   - 功能对比矩阵（我们vs他们，诚实且可辩护）
   - 定价比较分析
   - 需要强调的优势和需要回应的劣势
   - 与每个竞品相关的客户赢单故事
   - 带有具体反驳论点的异议处理话术

3. **按渠道的差异化文案**：COCO为每种用途生成竞品文案：
   - **网站**：对比落地页、"为什么选我们而非[竞品]"页面
   - **销售Deck**：竞品幻灯片及讲解要点
   - **邮件序列**：面向潜客的竞品差异化邮件
   - **广告文案**：竞品截流广告活动
   - **RFP响应**：针对具体评估标准的竞品定位

4. **异议处理脚本**：基于竞品的实际宣传和常见客户异议：
   - "他们说他们有[功能]。你们怎么比？"
   - "[竞品]便宜40%。我为什么要多付钱？"
   - "我看到[竞品]获了[奖项]。他们更好吗？"
   - 每个脚本包括：确认、重构、差异化、证据

5. **赢单/丢单分析支持**：COCO帮助结构化和分析赢单/丢单数据：
   - 针对每个竞品，赢单vs丢单的模式
   - 与赢单相关的消息主题
   - 赢单客户最常引用的竞品弱点
   - 基于趋势的消息调整建议

6. **语气校准**：竞品文案要把握分寸。COCO确保：
   - 差异化而非贬低（专业，不激进）
   - 声明可辩护且具体（不是模糊的最高级形容词）
   - 客户证据支撑定位声明
   - 符合比较广告的法规标准

**可量化的结果**

- **竞品对战胜率**：从34%提升到52%（+53%提升）
- **因竞品消息丢失的交易**：减少61%
- **作战卡更新频率**：从季度到每周
- **响应竞品发布的时间**：从3周缩短到24小时
- **销售在竞品场景中的信心**：+40%（自评调查）
- **竞品对比页面转化率**：+28%

**受益角色**

- **销售团队**：始终配备当前、准确的竞品信息
- **产品营销**：竞品定位保持新鲜，无需持续手动投入
- **营销领导者**：更快、更协调的竞品响应
- **竞品情报团队**：分析更快地转化为行动

## 实用提示词

**提示词 1: 竞品作战卡生成**
```
为与[竞品名称]竞争创建全面的销售作战卡。

我们的产品：[描述你的产品、核心功能、定价]
他们的产品：[描述你了解的他们的产品、功能、定价]
我们的目标客户：[描述理想客户画像]

生成包含以下部分的作战卡：
1. **快速概述**：一段话的竞品概况
2. **我们赢在哪里**：前3个差异化点及证据
3. **他们强在哪里**：诚实评估（让销售不会被打个措手不及）
4. **常见异议与回应**：客户考虑竞品时最常提出的5个异议，附具体反驳话术
5. **致命问题**：销售应该问客户的5个问题，能暴露竞品的弱点
6. **埋雷**：在竞品介入前，销售流程早期应该建立的定位
7. **客户赢单故事**：一个评估了双方最终选择我们的客户叙事模板

语言保持专业——差异化，而非贬低。
```

**提示词 2: 对比落地页文案**
```
为"[我们的产品] vs [竞品]"对比落地页撰写文案。

我们的产品：[核心功能、定价、理想客户]
他们的产品：[核心功能、定价、他们的定位]
我们的诚实优势：[列出4-5个]
他们的诚实优势：[列出2-3个——我们需要可信地承认这些]
访问这个页面的目标受众：[他们是谁，在研究什么]

页面结构：
1. 主标题和副标题（聚焦差异化，不激进）
2. 快速对比表（功能、定价、客服、集成）
3. 3个详细的"为什么选[我们的产品]"章节，含具体用例
4. 诚实的"什么情况下[竞品]可能更适合"章节（建立可信度）
5. 一个从竞品切换过来的客户评价
6. CTA部分

语气：自信且公平。我们要让读者因为我们诚实而信任我们，而不是因为我们贬低竞品。
```

**提示词 3: 竞品新功能发布的快速响应**
```
竞品[名称]刚刚发布了[描述他们的新功能/产品]。我们需要快速跨多个渠道响应。

他们的公告：[粘贴或总结他们的公告]
我们的产品对比情况：[我们有类似的吗？更好？不同方法？]
我们的实际优势：[他们仍然没有的我们做到的]

生成：
1. **内部Slack公告**给销售团队（发生了什么、该说什么、什么不该说）
2. **更新的作战卡章节**针对这个具体功能
3. **销售邮件模板**供销售发给正在评估竞品的潜客
4. **社交媒体回应**（如果合适——有时最好的回应是沉默）
5. **客户成功团队FAQ**（以防现有客户询问）

时间线：需要在24小时内发出。优先确保准确性，而非完美度。
```

**提示词 4: 赢单/丢单分析总结**
```
分析这些赢单/丢单数据点，识别改善竞品定位的模式。

近期竞品交易：

赢单：
1. [公司] - vs [竞品] - 赢单原因：[原因] - 交易金额：¥[X]
2. [公司] - vs [竞品] - 赢单原因：[原因] - 交易金额：¥[X]
[...继续]

丢单：
1. [公司] - vs [竞品] - 丢单原因：[原因] - 交易金额：¥[X]
2. [公司] - vs [竞品] - 丢单原因：[原因] - 交易金额：¥[X]
[...继续]

分析：
1. 按竞品的赢单/丢单模式
2. 最常见的赢单主题和丢单主题
3. 交易金额与赢单/丢单的关联
4. 消息缺口（我们应该说但没说的）
5. 产品缺口（导致我们丢单的功能）
6. 下个季度提升胜率的前3个可操作建议
```

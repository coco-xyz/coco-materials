# Zalo Personal — 部署与使用指南

将你的 COCO AI 员工通过个人账号连接到 Zalo。支持私聊、群聊、图片、文件、贴纸和表情回应 — 完整的 Zalo 体验。

## 前提条件

| 项目 | 说明 |
|------|------|
| Zalo 账号 | 一个安装了 Zalo 手机 App 的个人 Zalo 账号 |
| COCO AI 员工 | 已在 COCO Dashboard 中创建的实例 |
| 约 5 分钟 | 完成部署所需时间 |

**你不需要：**
- 任何服务器或技术基础设施
- API 密钥或开发者账号
- Zalo 官方账号（OA）注册
- 任何编程技能

::: warning 请使用专用 Zalo 账号
请使用一个**专用 Zalo 账号**来运行 Bot — **不要**使用你的主要个人 Zalo 账号。连接的账号将专门作为 Bot 使用。在同一账号上使用 Zalo Web 会导致 Bot 断开连接。
:::

## 部署

### 第 1 步：在 COCO Dashboard 中连接

1. 登录 [COCO Dashboard](https://coco.xyz/dashboard)
2. 进入渠道配置页面，选择 **Zalo Personal**
3. 点击 **Connect**（连接） — 系统将生成二维码（约需 30 秒，请耐心等待）

> **重要：** 等待期间请勿关闭页面或重复点击 — 这可能会触发重复请求。

### 第 2 步：用 Zalo 手机 App 扫描二维码

1. 打开手机上的 **Zalo** App
2. 点击主界面右上角的**扫码**图标
3. 将手机摄像头对准 Dashboard 上显示的二维码
4. 在手机上确认登录
5. 等待几秒钟完成连接

> **提示：** 如果二维码已过期，Dashboard 会自动刷新。扫描页面上显示的最新二维码即可。

### 第 3 步：开始使用

1. 扫码成功后，Dashboard 显示 **Connected**（已连接）
2. 用另一个 Zalo 账号搜索已连接的 Zalo 账号并发送消息
3. AI 员工立即回复 — 部署完成！

> **首条消息：** 第一个给 Bot 发送私聊消息的用户将成为 **Owner**（管理员）。Owner 始终拥有完整访问权限，不受策略设置限制。

## 使用技巧

### 私聊（DM）
直接给 Bot 账号发送消息即可，无需 @提及。

### 群聊
在群聊中 @提及 Bot：
```
@BotName 帮我总结一下这个文档
```

Owner 只需在新群中 @提及 Bot 即可激活 — 群组会自动注册。

### 智能模式 vs. 提及模式
群组可以在两种模式下运行：
- **提及模式**（默认）：只有 @提及 Bot 时才回复
- **智能模式**：Bot 接收所有消息，根据上下文决定是否回复

### 图片和文件
- **发送给 Bot：** 分享图片（JPG、PNG）和文件（PDF、DOCX 等）— Bot 可以阅读和分析
- **Bot 发送回复：** Bot 可以发送图片和文件给你
- 文件大小限制：接收最大 50 MB，发送最大 25 MB

### 贴纸和表情回应
- 在对话中可以发送和接收贴纸
- Bot 可以用表情回应消息（爱心、点赞、哈哈、哇、哭泣、生气）

### 语音消息
语音消息发送功能可用但需要公开托管的音频 URL。暂不支持接收语音消息。

## 访问控制

### 私聊访问
默认情况下，只有 Owner 可以给 Bot 发私聊消息。可通过以下模式配置访问权限：

| 模式 | 行为 |
|------|------|
| `owner` | 仅 Owner 可以私聊（默认） |
| `allowlist` | Owner + 指定用户可以私聊 |
| `open` | 任何人都可以私聊 |

### 群聊访问
群聊访问单独控制：

| 模式 | 行为 |
|------|------|
| `allowlist` | 仅已注册的群组有效（默认） |
| `open` | Bot 在任何群组中响应 @提及 |

每个群组可单独配置发送者限制（`allowFrom`）和模式选择（提及/智能）。

## 已知限制

| 限制 | 详情 |
|------|------|
| 单一 Web 会话 | 如果在同一账号上打开 Zalo Web 浏览器，Bot 会断开连接。每个账号只允许一个 Web 会话。 |
| 需要重新扫码 | 会话凭证可能会过期。发生时需返回 Dashboard 扫描新的二维码。 |
| 输入提示显示"正在电脑上输入" | 这是 Zalo 平台限制 — 输入指示器始终显示"正在电脑上输入"。 |
| 无富文本格式 | 所有发出的消息均为纯文本 — 不支持富文本格式化。 |
| 无法查看手机号 | Zalo 隐私限制导致 Bot 无法获取用户的手机号。 |

## 故障排除

| 问题 | 解决方案 |
|------|----------|
| Bot 不回复 | 在 COCO Dashboard 中检查连接状态。如已断开，重新扫描二维码。 |
| 二维码生成时间很长 | 首次生成可能需要 30-60 秒 — 这是正常现象。请勿关闭页面。 |
| Bot 意外断开连接 | 可能是同一账号上打开了另一个 Zalo Web 会话。关闭其他 Web 会话后重新连接。 |
| 无法发送文件给 Bot | 确保文件小于 50 MB。支持格式：图片、PDF、文档等。 |
| Bot 在群里不回复 | Bot 必须在群中被 @提及。Owner 需要先 @提及 Bot 以激活新群组。 |
| 需要多人使用 | 在 Dashboard 设置中将私聊访问策略改为 `allowlist` 或 `open` 模式。 |

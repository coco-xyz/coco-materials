# 流程自动化用例

负责自动化重复性工作流的AI员工。

| 用例 | 角色 | 功能 | 指标 |
|------|------|------|------|
| AI工单分类 | 客服 | 自动分类和路由支持工单 | 5分钟/工单→即时 |
| AI费用审核 | 财务 | 自动审核报销单对照政策 | 20分钟/份→即时 |
| AI CRM管家 | 销售 | 自动记录所有销售活动到CRM | 45分钟/天→0 |
| AI发票处理 | 财务 | 提取数据、匹配PO、路由发票 | 15分钟→30秒 |
| AI面试排期 | HR | 协调日历并发送邀请 | 45分钟→3分钟 |
| AI会议纪要 | 运营 | 录音、总结、分发行动项 | 30分钟→即时 |
| AI部署监控 | DevOps | 监控部署、异常自动回滚 | 手动→自动 |
| AI简历筛选 | HR | 批量按要求评分简历 | 3天→2小时 |
| AI报价计算 | 销售 | 套用定价规则生成报价单 | 2小时→10分钟 |

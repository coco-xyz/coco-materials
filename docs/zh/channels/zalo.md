# Zalo — 使用技巧

在 Zalo（越南最流行的即时通讯平台）上使用 COCO AI 员工。

## 配置 Zalo

### 准备工作

| 项目 | 说明 |
|------|------|
| COCO 账号 | 已注册并已付费/试用中 |
| Zalo Bot Token | 从 Zalo Bot 平台获取（见下方） |
| 约 5 分钟 | 完成部署所需时间 |

### 第 1 步：创建 Zalo 机器人

1. 访问 [Zalo Bot Platform](https://bot.zaloplatforms.com) 并登录
2. 点击 **Create Bot**（创建机器人）
3. 输入机器人名称和描述
4. 创建完成后，在机器人设置页面找到 **Bot Token**
5. 复制并保存 Token——下一步需要使用

> **注意：** Zalo Bot Token 使用 `numeric_id:secret` 格式。请妥善保管，不要泄露。

### 第 2 步：在 COCO Dashboard 中连接

1. 登录 [COCO Dashboard](https://coco.xyz/dashboard)
2. 打开 **Channels**（渠道）页面，找到 **Zalo** 卡片
3. 粘贴第 1 步获取的 **Bot Token**
4. 点击 **Connect**（连接）
5. 系统自动验证 Token 并完成连接

### 第 3 步：开始使用

1. 打开 Zalo，搜索你的机器人名称
2. 发送任意消息——AI 员工会立即回复
3. 部署完成！

> **提示：** Zalo Bot 默认使用轮询模式——无需公网 URL 或服务器。这是最简单的部署方式。

## 基本使用

### 私信（DM）

直接向机器人发送任意消息，机器人会立即回复。私信中无需 @mention。

```
你：帮我把这段话翻译成越南语
机器人：翻译结果如下……
```

### 群聊

将机器人添加到 Zalo 群组。在群聊中需 @mention 机器人来触发响应：

```
@你的机器人 帮我总结今天的讨论
```

> **注意：** 群组访问需要配置。默认情况下，只有机器人拥有者可以交互。请告知你的 AI 员工为特定群组启用群组访问。

## 支持的功能

| 功能 | 支持情况 |
|------|----------|
| 文本消息 | 支持 |
| 图片 | 通过 URL 发送 |
| 贴纸 | 发送和接收 |
| 文件 / PDF | 不支持 |
| 群聊 | 支持（需配置允许列表） |
| 输入状态指示 | 基础支持 |

## 消息投递模式

**轮询模式（默认）：** 无需额外配置。机器人自动检查新消息。适合大多数用户。

**Webhook 模式：** 适合希望实时接收消息的高级用户。需要公网 HTTPS URL。在 COCO Dashboard 的 Zalo 渠道设置中配置。

## 进阶技巧

**访问控制**：默认情况下，只有拥有者可以向机器人发消息。要允许其他人使用，告诉机器人："开启私信开放模式" 或 "将用户 123456 添加到允许列表"。

**群组配置**：告诉机器人添加群组，提供群组 ID 和所需模式（mention 或 open）即可。

**多渠道**：你可以将 Zalo 与其他渠道（Telegram、飞书等）同时连接——同一个 AI 员工会在所有渠道上响应。

## 故障排除

**机器人不响应？**
- 在 COCO Dashboard 中确认 Bot Token 是否正确
- 检查 COCO 订阅是否有效
- 尝试断开并重新连接渠道

**响应缓慢？**
- 轮询模式会定期检查消息——相比 Webhook 模式可能有轻微延迟
- 复杂请求需要更多时间（详细分析可能需要 30–60 秒）

**无法将机器人添加到群组？**
- 群组访问需要显式启用——请让你的 AI 员工配置群组权限
- 确认机器人已在 Zalo 平台上被添加到群组

**图片问题？**
- 发送图片需要公网 URL——不支持本地文件路径
- 接收的图片会自动下载并处理

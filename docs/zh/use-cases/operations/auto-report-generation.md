# 自动报告生成

> **岗位：** 行政助理 | **行业：** 通用 | **场景：** 工作流自动化

## 痛点

团队每人每天花30-60分钟手动编写日报。数据分散在项目工具、聊天记录和数据库中。报告质量参差不齐、经常延迟，而且没人喜欢写。

**算一笔账：** 10人团队每天浪费5小时——每月100+小时花在复制粘贴的总结上。

## AI员工解决方案

你的AI数字员工自动完成：
- **数据采集**：从项目管理工具、聊天记录、数据库和日历中提取数据
- **格式化**：按模板整理为统一、整洁的日报/周报
- **定时推送**：按计划发送到飞书、Telegram、Slack或邮箱（如每天下午6点）

## 使用前 vs 使用后

| 指标 | 使用前 | 使用后 |
|------|--------|--------|
| 每人耗时 | 30分钟/天 | 2分钟（仅需审阅） |
| 10人团队 | 每天浪费5小时 | 共计20分钟 |
| 报告质量 | 参差不齐，经常延迟 | 格式统一，准时交付 |
| 数据覆盖 | 不完整（手动回忆） | 完整（自动抓取所有数据源） |

## ROI

- **节省90%时间**用于报告编写
- **100%准时交付**——零延迟
- **更好的决策**——报告包含人工容易遗漏的数据

## 营销角度

几乎每家公司都需要日报/周报。这个痛点是普遍且深刻的。该用例在所有行业和岗位中都能引起共鸣。

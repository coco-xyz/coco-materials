# 用例 #263：AI 税务规划顾问

**角色**：财务专业人员 | **行业**：金融、制造、SaaS、房地产、专业服务、私募股权 | **任务**：合规、税务规划、财务战略

---
## 详细介绍

**痛点：企业平均多缴纳 18-25% 的税款，因为税务规划是被动的、碎片化的，而且在财年中启动得太晚**

税收是大多数企业损益表中金额最大的可裁量支出项——也是管理最为不善的一项。一家税前收入为 3000 万元、实际税率为 25% 的盈利中型企业，每年缴纳 750 万元的税款。税务专家估计，通过合理规划，其中 18-25% 的税负是可以合法避免的——这意味着每年有 135 万至 187.5 万元的税后现金，大多数企业只是因为税务规划流程是被动而非战略性的，而白白放弃了。

结构性问题在于时机。大多数企业将税务视为合规活动：申报纳税申报表，缴纳税款，对话结束，直到明年再说。等到企业的外部 CPA 机构提出税务规划对话时——通常是在第四季度或财年结束后——许多最有价值策略的规划窗口已经关闭。实体结构选择、退休计划设立、资本支出时机、研究抵免资格认定和州级关联规划，都需要在税年结束前采取行动，而不是之后。

对于在内部或通过单一外部 CPA 关系管理税务的中型企业而言，可用税务策略的广度确实令人望而生畏。联邦所得税与州及地方税、有任何跨境业务的公司的国际税、雇主税、销售和使用税，以及随每个预算周期变化的一系列税收抵免和激励措施相互交织。没有任何一位通才财务团队成员能够追踪所有这些内容。结果是企业依赖外部 CPA 处理合规事务，而错失了最能降低实际税率的主动顾问工作。

仅研究抵免一项就说明了被错过机会的规模。第 41 条研发税收抵免适用于任何开展符合条件研究的企业——不仅是科技公司，还包括开发新工艺的制造商、重新配方的食品公司、解决客户特定问题的工程公司。有资格申请抵免的公司平均每年获得 12 万至 25 万元的抵免。大多数合格公司要么根本不申请，要么申请的金额远低于其符合资格的金额，因为组织中没有人在全年追踪符合条件的活动。

**COCO 如何解决这个问题**

COCO 的 AI 税务规划顾问作为主动税务智能层发挥作用——持续识别规划机会、标记时机要求，并生成帮助财务团队和 CPA 合作伙伴更早做出更好税务决策的分析框架。

1. **年度税务状况分析和战略识别**：COCO 分析您公司的财务状况——收入、收益、实体结构、运营管辖区、资本结构、计划交易——并识别最有可能降低实际税率的税务规划策略，按估计影响和实施复杂性排序。

2. **研发税收抵免资格评估**：COCO 引导系统性识别和记录第 41 条（美国）或等效国际条款下的符合条件研发活动。它帮助按四步测试对活动进行分类，估算符合条件的支出，并生成 CPA 机构可用于支持抵免申请的文档框架。

3. **时机和加速扣除规划**：COCO 模拟时机决策的税务影响——奖金折旧选择、第 179 条费用化、加速扣除策略、预付费用选择——并根据当年和预计下一年的应税收入，推荐收入和扣除确认的最优时机。

4. **州及地方税关联分析**：COCO 将您公司的运营足迹——员工、承包商、销售、财产——与州关联门槛进行比对，以识别您可能存在未被认定的税务敞口的地方，以及需要州税登记或规划的地方。它还识别摊派规划可能降低整体州实际税率的州。

5. **实体结构和交易税务效率**：对于考虑并购、重组或新业务线的公司，COCO 分析不同结构方法的税务影响——C 型公司 vs. 穿透实体、资产 vs. 股权交易、国内 vs. 境外实体配置——并量化各备选方案之间的税后成本差异。

6. **年末税务预测和行动清单**：COCO 生成滚动年末税务预测——预计应税收入、预计税负、预计实际税率——并生成在财年结束前采取的实施已识别规划策略步骤的优先级行动清单。

**可量化成果**

- **实际税率降低**：借助 COCO 支持开展主动税务规划的企业报告平均实际税率降低 3-6 个百分点——对于税前收入 3000 万元的企业，每年价值 90 万至 180 万元
- **研发抵免获取**：此前未申请抵免的中型企业首年研发抵免申请平均为 18.5 万元——有适当文件记录后每年循环
- **完成的年末规划行动**：使用 COCO 的财务团队在财年结束前平均完成 7.2 项规划行动，而没有结构化支持的团队仅为 2.1 项
- **CPA 顾问价值**：携带 COCO 准备的税务状况分析参加 CPA 会议的企业报告，顾问会议效率提升 40%，由于准备的材料质量更高，专业费用也更低
- **规划周期启动日期**：使用 COCO 提示词后，平均规划周期在财年中提前 4.2 个月启动，而非被动的第四季度启动

**受益人群**

- **CFO 和财务总监**：获得结构化、主动的税务规划框架，而不是完全依赖外部顾问提出规划机会
- **税务经理**：携带情况分析、情景建模和文档框架（而非原始财务数据）与外部 CPA 机构更高效地合作
- **财务总监**：更早识别税务规划行动的应计和拨备影响，提高期末结账准确性
- **企业主（私营企业）**：获取穿透规划机会——QBI 扣除优化、S 型公司选择分析、退休计划策略——这些机会在没有主动支持的情况下通常被错过

## 实用提示词

**提示词 1：年度税务规划机会识别**
```
我想在财年结束前识别对我们公司可用的所有税务规划机会。

公司概况：
- 实体类型：[C 型公司/S 型公司/LLC/合伙企业]
- 财年结束日期：[日期]
- 行业：[描述]
- 运营州：[列出所有有员工、财产或销售的州]
- 预计当年税前收入：[$]
- 上年实际税率：[%]
- 当年预计实际税率（规划前）：[%]

业务活动：
- 收入来源：[描述——产品、服务、地理构成]
- 本年重大资本支出：[$，描述]
- 研发或产品开发活动：[是/否，如是请描述]
- 任何国际运营或交易：[是/否，如是请描述]
- 任何计划中的并购、重组或重大交易：[描述]
- 现有退休计划：[描述]

请：
1. 识别所有可能适用于我们状况的税务规划策略，按预计影响排序
2. 对每项策略提供：预计节税范围、所需行动、行动截止日期、复杂性（低/中/高）
3. 识别在我们财年结束前要采取的 5 项最高优先级行动
4. 标记我们业务活动可能产生我们应当解决的税务风险或敞口的领域
5. 识别我们在下次规划会议上应向 CPA 提出的 3-5 个问题
6. 为我们财年剩余月份创建税务规划日历
```

**提示词 2：研发税收抵免资格分析**
```
我想确定我们公司是否有资格申请研发税收抵免，并估算潜在收益。

公司概况：
- 行业：[制造/软件/食品/工程/其他]
- 年度收入：[$]
- 年度工资：[$]
- 年度耗材成本：[$]
- 年度承包商成本：[$]

可能符合研发条件的业务活动：
[描述任何涉及以下内容的活动：新产品开发、流程改进、软件开发、配方工作、工程问题解决、原型开发、测试和验证]

当前研发抵免状态：
- 我们目前是否申请研发抵免？[是/否]
- 如是，当前年度抵免金额：[$]
- 我们是否有符合条件活动的文件记录？[描述]

请：
1. 对每项描述的活动应用四步测试：具有技术性、消除不确定性、实验过程、符合条件的目的——评估哪些活动可能符合条件
2. 估算符合条件的研究支出（QRE）基础：与符合条件活动相关的工资、耗材、承包商成本
3. 使用普通抵免法（QRE 超出基础金额的 20%）和简化替代抵免法（QRE 超出前三年平均 QRE 50% 的 14%）计算预计联邦研发抵免
4. 识别我们运营州中可与联邦抵免叠加的州级研发抵免
5. 描述我们需要收集和保存的文件以支持抵免申请
6. 建议我们是否应聘请专业研发抵免机构，以及选择时应注意什么
```

**提示词 4：州税关联和摊派分析**
```
我需要分析我们的州税敞口并识别规划机会。

公司运营：
- 注册所在州：[州]
- 有员工的州：[列出，各州大致人数]
- 有实体财产/办公室的州：[列出]
- 有重大销售的州：[列出，各州大致年度收入]
- 有独立承包商的州：[列出]
- 有库存的州：[列出]

收入构成：
- 产品销售 vs. 服务收入：[%/%]
- 任何数字/SaaS 产品：[是/否]
- 任何知识产权许可：[是/否]

目前已在以下州登记：[列出]

请：
1. 评估关联风险：我们可能在哪些额外的州产生了我们目前未登记的所得税或销售税关联？
2. 估算每个潜在关联州的潜在未申报税务负债
3. 就我们存在历史敞口的州推荐关联审查和自愿披露策略
4. 分析摊派：根据我们的收入构成，哪种摊派方法（销售因素、三因素）产生最低的整体州实际税率？
5. 识别我们运营州中可用的任何州特定抵免或激励（研发抵免、就业创造抵免、企业区激励）
6. 按风险降低或节税潜力优先排序 3-5 项州税行动
```


# 视频脚本: AI API Doc Writer
时长: ~30秒
风格: 暖黄色调, COCO品牌

## 场景1 (0-5秒): 吸引注意
[淡入。暖黄色背景上的粗体文字。微妙的粒子动画。]
旁白: "我们的API有142个端点。文档覆盖了89个。那89个里面，至少30个已经过时——参数名称错误、缺少错误码、示例响应从去年Q2起就不再有效。开发者社区很沮丧。"文档说X但API返回Y"的支持工单占了我们工单量的23%"

## 场景2 (5-12秒): 痛点
[分屏：左侧展示疲惫的员工，右侧展示堆积如山的任务列表。数字向上滚动。]
旁白: "手动流程消耗了团队的时间和精力。每周都有大量时间浪费在本应几分钟就能完成的工作上。"

## 场景3 (12-22秒): 解决方案
[COCO logo出现。平滑过渡到仪表盘效果图。功能要点逐个动画展示。]
核心功能:
  - 读取代码库，从实际实现生成OpenAPI规范
  - 检测代码和现有文档之间的偏差，标记过时内容
  - 生成完整的端点文档：描述、参数、请求/响应示例、错误码、认证要求
旁白: "COCO的AI API Doc Writer端到端自动化整个流程。一次配置，持续运转。"

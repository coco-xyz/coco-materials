## 详细介绍

**痛点：SEO内容军备竞赛在人类速度下无法获胜**

SEO内容营销是一个有质量门槛的规模游戏。要排名竞争性关键词，你需要全面、权威、结构良好的内容——而且需要大量。Google的算法奖励主题权威性，这意味着需要用数十篇相互链接的文章深度覆盖一个主题群。

经济账很残酷。一篇高质量SEO文章需要多个专业技能：关键词研究、竞品分析、行业知识、文案写作、页面SEO优化和内链策略。每篇需要6-10小时的专业工作。按照有经验的内容营销人员每小时50-100美元计算，单篇成本在300-1000美元之间。

大多数公司能负担每周发布2-4篇文章。而他们在成熟市场的竞争对手有数千个已索引页面。差距不会缩小——只会扩大，因为更多的现有内容产生复合的权威优势。

**COCO如何解决**

COCO的AI SEO Content Writer将内容创作流水线从一个串行的、劳动密集型的过程转变为可扩展的系统。

1. **SERP分析引擎**：给定目标关键词，COCO：
   - 分析该关键词排名前10-20的页面
   - 提取它们的内容结构（标题、字数、涵盖话题）
   - 识别语义关键词和相关词（LSI关键词）
   - 发现内容缺口——排名靠前的结果没有充分涵盖的主题
   - 评估搜索意图（信息型、交易型、导航型）

2. **智能大纲生成**：基于SERP分析，COCO生成优化的大纲：
   - 涵盖排名靠前结果的所有内容（基本门槛）
   - 填补竞品遗漏的缺口（竞争优势）
   - 构建H2/H3标题层级，最大化SEO价值和可读性
   - 根据主题深度需求建议每个章节的字数目标
   - 将"People Also Ask"问题作为自然的子标题

3. **完整文章生成**：COCO撰写完整文章：
   - 自然的关键词密度（主关键词、次关键词、语义词）
   - 正确的标题层级和内容结构
   - 有吸引力的引言，包含钩子和清晰的价值主张
   - 实质性的正文章节，包含数据、案例和可操作建议
   - 有力的结论和CTA
   - 可扫描的排版（要点列表、编号列表、加粗关键短语）

4. **页面SEO优化**：每篇文章附带：
   - Meta标题（60字符，关键词优化，吸引点击）
   - Meta描述（155字符，包含关键词，驱动点击）
   - URL slug建议
   - 图片alt文本建议
   - Schema标记建议（FAQ、HowTo、Article）
   - 从现有内容库中推荐内部链接

5. **内容差异化**：COCO不产出千篇一律的内容：
   - 融入独特的数据点和统计数据
   - 建议竞品未涵盖的原创角度
   - 根据品牌调性指南调整语气和深度
   - 识别可以强化E-E-A-T信号的原创研究、调查或专家引用机会

6. **内容日历整合**：在规模化层面，COCO帮助规划：
   - 主题群映射（支柱页面+支撑文章）
   - 基于搜索量、难度和商业价值的关键词优先级
   - 老化文章的内容刷新计划
   - 域名级别的竞品内容缺口分析

**可量化的结果**

- **内容产量**：从每周2篇增至12+篇（6倍提升）
- **单篇成本**：从400美元降至90美元以下（降低78%）

## 实用提示词

**提示词 1: 基于关键词的完整SEO文章**
```
围绕目标关键词"[你的目标关键词]"撰写一篇全面的SEO文章。

写作前分析：
1. 该关键词的搜索意图（信息型/交易型/导航型）
2. 排名靠前的文章可能涵盖的内容
3. 能让这篇文章脱颖而出的内容缺口

文章要求：
- 字数：3000-4000字（中文）
- 包含针对相关关键词优化的H2和H3子标题
- 自然的关键词布局（主关键词出现在标题、H2、前100字和结论中）
- 至少包含3个数据点或统计数据并标注来源
- 添加FAQ部分，回答3-4个"大家还在问"类型的问题
- 对话式但权威的语气
- 包含读者可以立即执行的可操作建议

同时提供：
- Meta标题（30字以内）
- Meta描述（80字以内）
- 5个内部链接锚文本建议
- 3个建议配图及alt文本
```

**提示词 2: 竞品内容缺口分析**
```
我在"[你的主题领域]"这个话题上与以下域名竞争：
- [competitor1.com]
- [competitor2.com]
- [competitor3.com]

分析这些竞品的内容策略，识别：
1. 它们都涵盖的话题（我必须匹配的基本盘）
2. 只有1-2个涵盖的话题（差异化机会）
3. 它们都没有充分涵盖的话题（内容缺口=最大机会）
4. 它们可能遗漏的长尾关键词机会
5. 内容格式缺口（例如：它们有指南但没有对比文章）

对每个识别的缺口提供：
- 建议文章标题
- 目标关键词和预估搜索意图
- 简要大纲（3-4个H2标题）
- 优先级（高/中/低，基于搜索量潜力和难度）

输出为接下来8周的优先排序内容日历。
```

**提示词 3: 排名下降文章的内容刷新**
```
这篇文章发布于[X个月前]，排名正在下降。刷新它以提升表现。

当前文章：
[粘贴文章内容]

当前表现：
- 目标关键词：[关键词]
- 当前排名位置：[X]
- 最高排名位置：[X]（达成于[日期]）
- 月度自然流量：[X]（从[X]下降）

刷新要求：
1. 将所有统计数据和数据点更新到当前年份
2. 添加涵盖发布以来新出现话题的新章节
3. 用更强的钩子改进引言
4. 强化E-E-A-T信号（经验、专业、权威、信任）
5. 基于当前"大家还在问"结果添加新的FAQ问题
6. 针对新增搜索量的相关关键词进行优化
7. 用较新发布的内容改进内部链接

提供刷新后的文章和总结所有修改的变更日志。
```

**提示词 4: 主题群规划**
```
为"[你的核心主题]"构建全面的主题群策略。

创建：
1. **支柱页面**：涵盖整个主题的3000+字综合指南
   - 含H2/H3结构的大纲
   - 目标主关键词和次关键词

2. **支撑文章**（10-15篇）：每篇针对一个特定长尾关键词
   - 文章标题
   - 目标关键词
   - 字数建议
   - 如何链接回支柱页面
   - 简要大纲（3个H2标题）

3. **内链地图**：所有内容如何相互连接

4. **发布顺序**：为最大化SEO效果的最优发布顺序

我的网站域名权重大约是[X]。重点关注与此权重水平匹配难度的关键词。
```

```
为以下现有页面优化meta标题和描述。每个meta标题不超过30字，meta描述不超过80字。两者都需要自然包含目标关键词，并且足够吸引人以提升点击率。

需要优化的页面：
1. URL: [url] | 当前标题: [标题] | 目标关键词: [关键词]
2. URL: [url] | 当前标题: [标题] | 目标关键词: [关键词]
3. URL: [url] | 当前标题: [标题] | 目标关键词: [关键词]
[...继续列出所有页面]

对每个页面提供：
- 优化后的meta标题（附字数统计）
- 优化后的meta描述（附字数统计）
- 修改理由
- 预估CTR提升潜力（低/中/高）
```

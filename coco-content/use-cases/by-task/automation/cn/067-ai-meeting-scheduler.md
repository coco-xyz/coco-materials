# 用例 #067: AI会议调度助手

**运营、产品经理 | 行业：企业、SaaS | 任务：日程调度、流程自动化**

---
## 详细介绍

**痛点分析：安排会议每周浪费管理者5小时以上**

哈佛商业评论发现，高管平均每周花23小时开会——而安排会议本身又消耗5小时以上的邮件往返、日历协调和时区转换。对于一个10人的领导团队来说，每周集体损失50小时的生产力，仅仅用在找到大家都有空的时间上。跨职能会议、外部利益相关者和跨多个时区的全球团队让问题呈指数级增长。

隐性成本远不止浪费时间。当日程安排太慢，关键决策就会被拖延。本应周一举行的产品发布评审被推到周四，因为三位VP的日历对不上。一个紧急的客户会议要花48小时来安排，因为客户经理在和客户助理打邮件乒乓球。研究显示，71%的职场人士认为低效的会议安排是工作中的主要挫折源。

**COCO如何解决**

COCO的AI会议调度助手将这种混乱转变为流畅的智能工作流。以下是分步流程：

1. **日历智能分析**：COCO集成所有主流日历平台（Google Calendar、Outlook、Calendly），为每位参会者构建实时可用性地图。它不仅了解空闲时段，还理解会议模式——谁有连续会议疲劳，谁偏好上午做深度工作，哪些时间块是灵活的还是真正不可用的。

2. **自然语言调度**：团队成员无需使用复杂的调度界面，只需告诉COCO："下周安排一个45分钟的产品评审，参加者有Sarah、Mike和设计团队，最好是周二或周三下午。"COCO处理所有后勤工作。

3. **智能时间优化**：COCO不只是找到任意可用时段——而是找到最优时段。它考虑全球团队的时区公平性（轮换早/晚会议）、无会议时间块的保护、面对面会议间的通勤时间以及精力管理（避免连续马拉松会议）。

4. **自动化协调**：对于外部会议，COCO通过邮件直接与参与者沟通，按相互便利性排序提供多个时间选项。它处理来回协商、发送日历邀请、预订会议室并自动设置视频会议链接。

5. **冲突解决与重新安排**：当发生冲突时（重复预订、优先会议、取消），COCO主动建议替代方案。如果关键会议需要改期，它分析对所有参会者日程的连锁影响，提出干扰最小的改期方案。

6. **会前准备自动化**：在每次会议前，COCO汇总相关背景：上次会议纪要、待办事项、相关文档以及首次会面参与者的背景介绍。参会者在开始前30分钟收到简要的准备资料包。

**可衡量的成果**

使用COCO AI会议调度助手的团队报告：
- **会议安排后勤时间减少87%**
- **平均会议设置时间从34次邮件往返降到零手动操作**
- **跨时区会议安排速度提升3倍，通过自动公平轮换**
- **会前准备时间减少60%，通过自动生成的背景资料包**
- **会议取消率降低23%，通过主动冲突检测和解决**

**谁能受益**

- **行政助理**：消除数小时的日历俄罗斯方块和邮件乒乓球，专注于更高价值的行政支持工作
- **项目经理**：在几秒内而非几天内安排冲刺评审、利益相关者同步和跨团队会议
- **销售团队**：通过智能可用性共享即时预订客户会议，减少销售周期中的日程摩擦
- **远程团队负责人**：通过自动轮换追踪确保全球团队会议的时区公平分配

## 实用提示词

**提示词 1：优化每周会议日程**
```
审查团队当前的每周会议安排并优化生产力。当前定期会议：

[粘贴定期会议列表：名称、时长、参会者、当前日期/时间]

团队信息：
- 12人跨3个时区(EST、PST、GMT)
- 核心协作时间：EST 10am-2pm
- 团队偏好：当地时间9点前不开会，希望周五无会议

优化目标：
1. 最小化上下文切换(将相似会议批量安排)
2. 为每个团队成员保留每日2小时深度工作时间块
3. 跨时区会议的公平时区轮换
4. 通过识别冗余或可合并的会议减少总会议时间
5. 为不需要实时讨论的会议建议异步替代方案

输出：附带每项变更理由的优化后周历建议。
```

**提示词 2：创建会议类型模板**
```
为我们的组织创建标准化会议模板。需要以下会议类型的模板：

1. 冲刺规划(60分钟，双周)
2. 1:1经理签到(30分钟，每周)
3. 客户季度业务回顾(60分钟)
4. 全员公司更新(45分钟，每月)
5. 事件响应作战室(临时，紧急)

每个模板定义：
- 必需vs.可选参会者(按角色)
- 议程结构与时间分配
- 会前准备清单
- 需要准备的文档/数据
- 会后交付物(行动项格式、纪要分发)
- 安排约束(首选日期/时间、频率)
- 成功标准(如何判断会议是否高效)
```

- 会议类型分布：[列表]
- 缺席/取消率：[百分比]
- 超时会议：[百分比]

分析报告：
1. 会议成本分析(参会者工时×平均时薪)
2. 参会人数过多的会议(识别不需要参加的人)
3. 持续超时的会议——需要更长时段还是更好的议程？
4. 时段分析：哪些时段参与度最高vs.取消最多
5. 应改为异步的定期会议(低参与度、仅信息分享)
6. 每个团队的建议会议预算(每周最大会议小时数)

[附上会议分析数据]
```

```

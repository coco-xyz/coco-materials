# Zalo Personal — 使用技巧

通过个人 Zalo 账号使用 COCO AI 员工——完整支持群聊、文件、贴纸和表情回应。

## 配置 Zalo Personal

### 准备工作

| 项目 | 说明 |
|------|------|
| COCO 账号 | 已注册并已付费/试用中 |
| Zalo 账号 | 一部已安装并登录 Zalo 的手机 |
| 约 5 分钟 | 完成部署所需时间 |

::: warning 请使用专用 Zalo 账号
请使用一个**专用 Zalo 账号**作为机器人——**不要**使用你的个人 Zalo 账号。连接的账号将专门用作机器人。在浏览器中使用 Zalo Web 会断开机器人连接。
:::

**无需 API 密钥或开发者账号**——只需扫描二维码即可。

### 第 1 步：在 COCO Dashboard 中连接

1. 登录 [COCO Dashboard](https://coco.xyz/dashboard)
2. 打开 **Channels**（渠道）页面，找到 **Zalo Personal** 卡片
3. 点击 **Connect**（连接）——系统会生成二维码（可能需要约 30 秒）

### 第 2 步：扫描二维码

1. 在手机上打开 **Zalo**
2. 前往 **设置 → 二维码扫描器**（或点击二维码图标）
3. 扫描 Dashboard 上显示的二维码
4. 在手机上确认登录

### 第 3 步：开始使用

1. Dashboard 显示连接状态为已激活
2. 从另一个账号向已连接的 Zalo 账号发送消息
3. AI 员工响应——部署完成！

> **注意：** Zalo 同时只允许一个网页端会话。如果在浏览器中打开 Zalo Web，机器人会话将断开。如需重新连接，请从 Dashboard 重新扫码。

## 基本使用

### 私信（DM）

直接向机器人的 Zalo 账号发送任意消息，无需特殊命令。

```
你：帮我总结这份文件
机器人：以下是总结……
```

### 群聊

将机器人的 Zalo 账号添加到群组。在群聊中 @mention 机器人来触发响应：

```
@机器人名 帮我草拟回复给这个客户
```

> **提示：** 群组支持两种模式——**mention**（仅在被 @mention 时响应）和 **smart**（查看所有消息，在相关时响应）。可以让你的 AI 员工配置首选模式。

## 与 Zalo Bot 平台的对比

| | Zalo Personal | Zalo Bot 平台 |
|---|---|---|
| 配置方式 | 扫描二维码 | 从开发者平台获取 Bot Token |
| 群聊 | 完整支持 | 仅允许列表 |
| 文件和 PDF | 支持 | 不支持 |
| 表情回应 | 支持 | 不支持 |
| 图片 | 文件发送/接收 | 仅 URL |
| API 类型 | 非官方 (zca-js) | 官方 API |
| 风险 | 账号可能被限制 | 稳定，官方支持 |

如需更丰富的功能（群聊、文件、表情回应），选择 **Zalo Personal**。如需稳定性和官方 API 支持，选择 **Zalo Bot 平台**。

## 进阶技巧

**访问控制**：默认情况下，只有拥有者可以向机器人发消息。要允许其他人使用，告诉机器人："开启私信开放模式" 或 "将用户 123456 添加到允许列表"。

**群组模式**：让机器人将群组设为 "smart 模式"——它会查看所有消息，仅在相关时响应，无需每次都 @mention。

**发送文件**：直接在聊天中分享 PDF、文档和图片。机器人会处理并回复文件内容。

**表情回应**：机器人可以用表情回应消息。告诉它对特定消息做出回应，或设置自动回应。

**多渠道**：将 Zalo Personal 与其他渠道（Telegram、飞书、WhatsApp 等）同时连接——同一个 AI 员工会在所有渠道上响应。

## 故障排除

**机器人掉线？**
- Zalo 同时只允许一个网页端会话——如果你在浏览器中打开了 Zalo Web，机器人就会断开
- 返回 Dashboard 重新扫描二维码即可重新连接

**机器人在群聊中不响应？**
- 确认机器人账号已被添加到群组
- 在 mention 模式下必须 @mention 机器人
- 请让你的 AI 员工确认群组是否已配置

**二维码不显示？**
- 首次生成二维码可能需要 30–60 秒——请勿关闭页面
- 如果仍未出现，尝试刷新 Dashboard 页面

**响应缓慢？**
- 复杂请求需要更多时间（详细分析可能需要 30–60 秒）
- 检查 COCO 订阅是否有效

**会话过期？**
- 如果手机长时间离线，Zalo 会话可能过期
- 从 Dashboard 扫描新的二维码即可重新连接

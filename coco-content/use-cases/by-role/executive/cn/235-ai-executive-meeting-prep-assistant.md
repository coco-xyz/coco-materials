# Use Case #235: AI Executive Meeting Prep Assistant

**Role**: 行政助理 / 首席幕僚长 / C级高管 | **Industry**: 企业、专业服务、金融服务、科技 | **Task**: 会议准备、高管简报、日程安排、干系人管理

---
## 详细介绍

**痛点：高管在高风险会议前准备不足，因为充分准备需要3-4小时而没人有这个时间**

关于会议准备的研究结论是明确的：带着对与会者的当前情报、对关系历史的背景了解、对对方诉求和原因的清晰认知，以及准备好的谈话要点进入高风险会议的高管，在效果上有着可量化的优势——他们更快做出决策、建立更强的融洽关系，会后也有更清晰的后续步骤。问题在于，对单次重要外部会议进行彻底准备——客户季度业务回顾、董事会成员例行沟通、与新对口人的合作讨论、关键候选人面试——若做到位需要3至4小时。一位每周有8至12次重要外部会议的CEO或VP根本无力投入这么多时间。

实际情况是分层的准备失败。对于最重要的会议——董事会演示、重大客户谈判、高知名度候选人——高管会专门安排准备时间并进行充分准备。对于中间层级的会议——定期客户例会、合作伙伴会议、行业同伴电话——准备是临时和不一致的。高管在前往会议的路上浏览一下LinkedIn资料，在电话开始前30秒查阅上次对话的笔记，或者依靠记忆回忆6个月前发生的事情。对于最低层级的会议——预定例会、供应商审查、内部干系人会议——通常完全没有准备。结果是高管在会议中被动应付而非主导议程，关系背景随时间流逝，机会因为高管没有状态识别而错失。

这个问题随着行政助理（EA）或首席幕僚长角色的介入而加剧。高效组织的EA会为重要会议准备简报文件——但每份文件是1-2小时的工作，支持繁忙高管日程的EA只能覆盖20-30%的会议并提供适当的简报材料。其余的均未覆盖。EA最终将有限的调研时间花在最重要的会议上，而数十个中等重要的会议在没有足够背景的情况下发生。

**COCO如何解决**

COCO的AI高管会议准备助理自动为每场预定会议准备全面的简报包——将LinkedIn、公司新闻、CRM历史、之前的会议记录和高管的既定目标综合成结构化的会前简报，只需数分钟。

1. **自动与会者情报收集**：从可用来源调研每位与会者并综合出简洁的画像。
   - LinkedIn：角色、任期、职业背景、近期动态和帖子、共同联系人、教育背景
   - 公司背景：近期新闻、融资事件、领导层变动、财报公告、战略举措
   - CRM历史：关系时间线、之前的商机阶段、任何未结订单、支持互动
   - 之前的会议记录：历次对话的关键主题、双方做出的承诺、未解决的事项
   - 社交信号：近期发表的文章、会议演讲、公开声明——表明当前优先事项或关切

2. **关系历史综合**：从所有可用记录重构高管与外部对应方的关系脉络。
   - 总结关系时间线：我们认识这个人多久了，关系如何建立，我们一起做过什么
   - 识别关系健康信号：近期互动的频率和基调、响应性、任何摩擦点
   - 标记待办事项：之前会议中做出的承诺尚未跟进、提出的问题从未回答
   - 突出"上次对话背景"，让高管能够自然引用并展示持续的关注

3. **议程情报与目标对齐**：分析既定会议议程，结合可用背景呈现可能的隐性议程和未言明的目标。
   - 根据对方当前情况（近期公司新闻、职业阶段、已知举措）推断对方可能想从会议中得到什么
   - 识别高管目标与对方可能目标之间的潜在错位
   - 标记任何敏感话题——基于近期事件可能意外出现的内容——并为高管提供背景准备
   - 根据关系历史和当前背景，生成开场会议的推荐"框架"

4. **谈话要点与问题准备**：生成针对特定会议量身定制的即用型谈话要点和战略问题。
   - 产出高管在这次会议中想要传达的3-5条关键信息，针对特定受众进行框架定制
   - 生成4-6个高价值问题，旨在揭示对方的优先事项、关切和决策标准
   - 根据会议背景，为可预见的阻力准备潜在的异议回应
   - 包含"关系强化剂"——关于与会者近期工作或成就的具体观察，可以自然引用

5. **会后捕获与跟进落实**：实现快速捕获会议结果并自动生成跟进行动。
   - 用会前背景填充结构化会后模板，使记录更快
   - 从会议记录中提取并分配跟进行动，附责任人和截止日期
   - 用会议中捕获的新背景更新关系记录
   - 根据会议结果安排适当的后续联系

**可量化的结果**

- **会议准备时间**：每次重要会议从45-90分钟缩减至8-12分钟的审阅——节省6-7倍时间
- **EA简报覆盖率**：使用COCO的EA将覆盖率从20-30%的会议提升至85-95%的会议
- **会后跟进落实率**：高管报告在使用结构化捕获相比非结构化笔记时，后续行动完成率提高67%
- **关系延续性评分**：使用COCO的高管在78%的回访会议中引用了之前的对话背景，而未使用结构化准备的只有31%——可量化地改善关系质量
- **会议目标达成率**：高管在有适当会前简报时，"在此会议中实现主要目标"的自报率从54%提升至71%

**受益角色**

- **C级高管（CEO、CFO、销售VP等）**：每次会议——不仅仅是最重要的——都带着当前情报和清晰议程进入，将每次互动转化为关系建设机会
- **行政助理**：为完整的高管日程而非有限的子集产出全面的简报包，将EA角色从后勤协调人提升为战略赋能者
- **首席幕僚长**：为高管准备董事会会议、重要外部关系和战略合作伙伴对话，所需情报深度以前需要一整天的准备时间
- **业务拓展和合作团队**：带着商业关系的完整背景、竞争情报和对齐的战略目标进入每次外部会议

## 实用提示词

**提示词1：会前情报简报**
```
我在[X小时/天]后有一个重要会议，需要一份全面的简报包。

会议详情：
- 会议类型：[客户季度回顾/新业务拓展/董事会成员例行沟通/合作讨论/关键候选人面试/投资人更新/其他]
- 日期和时长：[日期、开始时间、时长]
- 我在这次会议中的角色：[我是谁，我想实现什么]

与会者：
1. [姓名、职位、公司] — [你对他们的任何了解]
2. [姓名、职位、公司] — [任何了解]
[添加所有与会者]

我能提供的背景：
- 我们如何认识这些人：[关系历史——多长时间、如何相识、之前的互动]
- 我对他们公司/情况的了解：[任何相关新闻、挑战或背景]
- 我们之前的会议记录（粘贴或总结）：[之前的对话内容]
- 我在这次会议中想实现的目标：[我的主要目标、次要目标]
- 任何敏感话题或顾虑：[我担心的或想谨慎处理的内容]
- CRM备注/邮件背景（可选）：[粘贴相关邮件或CRM备注]

请产出：
1. 与会者画像：关键背景、近期动态、对此次会议的可能优先事项
2. 关系摘要：你如何描述这段关系，最重要的近期背景是什么？
3. 会议情报：这个人/公司可能想从这次会议中得到什么？是否有需要注意的隐性议程？
4. 推荐的会议框架：鉴于背景，我应该如何开场？
5. 我应该传达的5个谈话要点——按重要性排序
6. 我应该提出的4个推进我目标的问题
7. 我在会议前应该采取的任何会前行动
```

**提示词2：董事会成员或投资人会前简报**
```
我正在为与[姓名/机构]的[董事会会议/投资人例会/LP更新]做准备。

关系背景：
- 董事会成员/投资人：[姓名、机构、加入我们董事会/投资时间]
- 他们的背景：[关于他们是谁的相关背景]
- 我们的关系历史：[关系如何演变]
- 上次互动：[时间、讨论了什么、任何未解决的事项]

当前公司背景（要分享或讨论的）：
- 我们的当前指标：[关键KPI——营收、增长率、烧钱率、达成或错失的关键里程碑]
- 进展顺利的：[前2-3项成果]
- 面临挑战的：[前1-2项我需要讨论的挑战]
- 我需要他们提供的：[具体请求——介绍、关于X的建议、关于Y的董事会批准]

已知的关切：
- 他们之前提出的、我应该有所准备的话题：[列表]
- 任何可能引发问题的指标：[标记任何低于预期的内容]

请产出：
1. 这位干系人最关心的是什么，以及如何专门为他们框架对话
2. 推荐的议程顺序：先涵盖什么、如何安排讨论顺序
3. 挑战性话题的谈话要点——如何结合背景和前进计划来呈现
4. 准备好的具体请求，用清晰、自信的2-3句话表述
5. 向他们提问的问题，使这成为双向关系对话，而非单纯的汇报
6. 这次会议中要回避或谨慎处理的内容
```

**提示词3：新联系人会议准备**
```
我要第一次与某人会面，需要做好准备以留下强烈的第一印象并推进特定目标。

会议背景：
- 我要会见的人：[姓名、职位、公司]
- 这次会议是如何安排的：[共同介绍、主动联系、外联、会议等]
- 我对他们的了解：[粘贴他们的LinkedIn简介或你拥有的任何背景]
- 他们的公司：[公司做什么，近期新闻（如有）]
- 我在这次会议中的目标：[我希望此次对话的结果是什么]

我的背景（他们可能已经了解的）：
- 我的角色和公司：[简要描述]
- 为什么这次会议对他们有价值：[从他们的角度，这次对话的价值是什么？]

请：
1. 识别3件关于此人的事情，我可以自然引用以展示我做了功课并建立融洽关系
2. 他们可能想从这次会议中得到什么？他们可能的"对我有什么好处"是什么？
3. 这次对话最好的开场是什么——我应该如何框架我们为何在谈话？
4. 4-5个开放式探索性问题，能揭示他们的优先事项和挑战
5. 我如何从探索过渡到推进我的目标——什么是合适的时机和措辞？
6. 这次会议结束时我应该提议什么后续行动来创造前进动力？
```

**提示词4：会后跟进与后续步骤捕获**
```
我刚刚完成了一场会议，需要快速记录关键成果并制定跟进计划。

刚刚发生的会议：
- 我与谁会面：[姓名和背景]
- 我们讨论了什么（粗略笔记——粘贴或描述）：[任何形式的会后笔记]
- 关键结果：[决定了什么、达成了什么一致、得出了什么结论]
- 会议基调：[进展如何——好于预期/符合预期/比预期更有挑战性]

会前简报背景（如有）：
- 我进入时的既定目标：[我想实现什么]
- 之前的关系背景：[会议前的相关内容]

请：
1. 用3-5个要点总结会议：发生了什么、决定了什么
2. 我实现了我的主要目标吗？如果没有，发生了什么，前进的路径是什么？
3. 提取所有行动项——我的和他们的——附明确的责任人和推荐截止日期
4. 起草一封24小时内发送的跟进邮件：确认讨论内容、记录承诺、提议后续步骤
5. 根据我在会议中了解到的，我应该在关于此人的关系笔记中更新什么？
6. 这次会议中有什么内容应该改变我今后的方式或预期吗？
```

**提示词5：每周高管日程回顾与优先级排序**
```
我想回顾我即将到来一周的会议，确保我做好准备，并且时间分配到了正确的优先事项上。

我即将到来的一周：
[以任意格式列出你本周的会议——周一到周五，包括：会议标题、与会者、时长、既定目的]

我本周的首要任务：
- 优先事项1：[什么最重要——一笔交易、一段关系、一个需要达成的决定]
- 优先事项2：[第二优先]
- 优先事项3：[第三优先]

当前背景：
- 我工作中当前正在发生的关键事情：[简要情况总结]
- 我特别担心的任何会议或关系：[标记任何问题]

请：
1. 评估每个会议：高价值（战略性，必须准备）、中等价值（常规，轻度准备）、低价值（可能不需要我亲自参与或可以异步处理）
2. 识别我应该最彻底准备的会议及其原因
3. 对于最高优先级的2-3个会议，概述我应该做什么准备以及我需要什么信息
4. 本周是否有任何会议我可能会被问到应该预先准备答案的问题？
5. 识别任何日程冲突或连续会议，使我没有恢复时间——是否有任何应该重新安排？
6. 本周哪一场会议，充分准备对我的优先事项ROI最高？
```

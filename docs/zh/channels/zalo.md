# Zalo — 部署与使用指南

将你的 COCO AI 员工连接到 Zalo，使用官方 Bot Platform API。支持私聊文字、图片和贴纸。

## 前提条件

| 项目 | 说明 |
|------|------|
| Zalo 账号 | 一个用于创建 Bot 的个人 Zalo 账号 |
| COCO AI 员工 | 已在 COCO Dashboard 中创建的实例 |
| 约 5 分钟 | 完成部署所需时间 |

**你不需要：**
- 任何服务器或技术基础设施
- 公网域名或 SSL 证书
- Zalo 官方账号（OA）注册
- 任何编程技能

---

## 部署

### 第 1 步：在 Zalo Bot Platform 创建 Bot

1. 访问 [Zalo Bot Platform](https://bot.zaloplatforms.com) 并用你的 Zalo 账号登录
2. 点击 **Create Bot**（创建 Bot）
3. 输入 Bot 名称（例如 `COCO AI`）和描述
4. 创建完成后，你会看到 **Bot Token**（格式：`数字ID:密钥`）
5. **复制并保存此 Token** — 下一步会用到

> **重要：** Bot Token 是你的 Bot 的唯一凭证，请勿分享给他人。

### 第 2 步：在 COCO Dashboard 中连接

1. 登录 [COCO Dashboard](https://coco.xyz/dashboard)
2. 进入渠道配置页面，选择 **Zalo**
3. 粘贴第 1 步获取的 **Bot Token**
4. 点击 **Connect**（连接） — 系统将验证 Token 并完成连接

### 第 3 步：开始使用

1. 在手机或电脑上打开 Zalo
2. 搜索你的 Bot 名称
3. 发送任意消息 — AI 员工立即回复
4. 部署完成！

> **首条消息：** 第一个给 Bot 发送私聊消息的用户将成为 **Owner**（管理员）。Owner 始终拥有完整访问权限，不受策略设置限制。

## 使用技巧

### 私聊（DM）
直接给 Bot 发送消息即可，无需 @提及。

### 图片
- **发送图片给 Bot：** 在聊天中分享图片 — Bot 可以查看和分析（最大 10 MB）
- **Bot 发送图片：** Bot 可以通过公开 URL 发送图片

### 贴纸
在与 Bot 的对话中可以发送和接收贴纸。

### 语音消息
Zalo Bot Platform 不支持语音消息。

---

## 访问控制

### 私聊访问
默认情况下，只有 Owner 可以给 Bot 发私聊消息。可通过以下模式配置访问权限：

| 模式 | 行为 |
|------|------|
| `owner` | 仅 Owner 可以私聊（默认） |
| `allowlist` | Owner + 指定用户可以私聊 |
| `open` | 任何人都可以私聊 |

## 功能列表

| 功能 | 状态 |
|------|------|
| 私聊消息 | 可用 |
| 接收图片 | 可用 |
| 发送图片（通过 URL） | 可用 |
| 贴纸（收发） | 可用 |
| 输入状态指示 | 可用 |
| 主动消息推送 | 可用 |
| 群聊 | 不可用 |
| 文件附件 | 不可用 |
| 语音消息 | 不可用 |
| Markdown 格式化 | 不可用 |

## 已知限制

| 限制 | 详情 |
|------|------|
| 无群聊功能 | Bot Platform API 仅支持私聊。群聊功能需要另外注册 Zalo 官方账号（OA）。 |
| 无文件附件 | Bot 无法发送或接收文件（PDF、文档等）。 |
| 无富文本 | 所有发出的消息均为纯文本 — 不支持格式化。 |
| 图片需要公开 URL | Bot 只能通过公开可访问的 HTTPS URL 发送图片。 |
| 引用回复不显示 | API 接受回复引用参数，但 Zalo 不会在界面上显示引用效果。 |

## 故障排除

| 问题 | 解决方案 |
|------|----------|
| Bot 不回复 | 检查 Bot Token 是否正确。在 COCO Dashboard 中确认连接状态。 |
| 无法发送图片给 Bot | 确保图片小于 10 MB。支持格式：JPG、PNG。 |
| Bot 发送的图片不显示 | 发出的图片必须托管在公开可访问的 HTTPS URL 上。 |
| 需要多人使用 | 在 Dashboard 设置中将私聊访问策略改为 `allowlist` 或 `open` 模式。 |

# 视频脚本: AI Job Description Writer
时长: ~30秒
风格: 暖黄色调, COCO品牌

## 场景1 (0-5秒): 吸引注意
[淡入。暖黄色背景上的粗体文字。微妙的粒子动画。]
旁白: "我们分析了为什么我们的工程职位需要47天才能填补，而行业中位数是36天。瓶颈不是寻源或面试——是申请量。Pipeline很薄因为合格候选人不申请。当我们审计职位描述时，原因变得明显了。它们读起来像合规文件，不像招聘广告"

## 场景2 (5-12秒): 痛点
[分屏：左侧展示疲惫的员工，右侧展示堆积如山的任务列表。数字向上滚动。]
旁白: "手动流程消耗了团队的时间和精力。每周都有大量时间浪费在本应几分钟就能完成的工作上。"

## 场景3 (12-22秒): 解决方案
[COCO logo出现。平滑过渡到仪表盘效果图。功能要点逐个动画展示。]
核心功能:
  - 分析实际角色需求，将真正的要求和加分项分开
  - 撰写以影响力和成长开头的描述，而非需求清单
  - 运行包容性语言分析：移除性别化语言、过多要求和排斥性行话
旁白: "COCO的AI Job Description Writer端到端自动化整个流程。一次配置，持续运转。"

## 场景4 (22-28秒): 成果
[影响指标以计数动画展示。绿色向上箭头。]
旁白: "申请量：+64%。女性申请者：+41%。Pipeline中的多元化候选人：+38%。招聘周期：从47天降至31天。同样的职位，更好的描述"

# Writing & Content Use Cases

AI employees that write, edit, and generate content.

| Use Case | Role | What It Does | Metric |
|----------|------|-------------|--------|
| AI SEO Content Writer | Marketing | Researches keywords, writes optimized blog posts | 4h → 20min/article |
| AI API Doc Writer | Dev | Generates API docs from codebase with examples | 1 week → 2 hours |
| AI Proposal Generator | Sales | Creates customized sales proposals from templates | 3h → 15min |
| AI Job Description Writer | HR | Writes inclusive, SEO-optimized job listings | 2h → 10min |
| AI Newsletter Curator | Marketing | Curates stories, writes summaries, formats newsletter | 5h/week → 30min |
| AI Follow-up Writer | Sales | Drafts personalized post-meeting emails | 30min → 2min |
| AI Ad Copy Generator | Marketing | Generates hundreds of A/B test ad variants | 2 days → 10min |
| AI Knowledge Base Builder | Support | Turns resolved tickets into FAQ articles | 2 days → 1 hour |

# #356 — AI 续约谈判准备助手

**角色**: 客户成功
**行业**: SaaS / 科技
**任务**: 分析
**标识符**: `ai-renewal-negotiation-prep-assistant`

---

## 简介

**痛点：续约对话在没有赢得对话的证据情况下进行**

续约谈判是客户成功职能中风险最高的对话——它们决定了上一年认可的收入是否持续、是否增长，以及在许多情况下整个关系是否延续。然而大多数 CS 经理以一种具体且影响重大的方式进入续约对话——准备不足：他们没有做分析工作来理解自己究竟在捍卫什么价值，以及究竟在应对什么风险。结果是谈判动态默认为价格讨论而非价值捍卫，因为 CSM 无法自信地阐明产品在续约期间交付的具体、量化影响。

证据缺口是防御性续约对话的主要原因。一位带着完整记录的账户健康分析、与客户具体业务目标挂钩的清晰投资回报指标、显示部署扩展的使用趋势数据以及突出差异化的竞争背景进入续约会议的 CS 经理，与一位带着通用幻灯片和希望客户没有在寻找替代方案进入会议的 CS 经理，正在进行根本不同的谈判。这两种立场之间的差异不是才能——而是准备，具体地说是构建全面的、基于证据的续约案例的分析工作。

竞争情报缺口加剧了这个问题。在大多数竞争市场中，续约周期与积极的竞争外联期相吻合——竞争对手知道 SaaS 合同通常何时到期，并据此构建他们的出站销售动作。不知道哪些竞争对手正在与其续约账户进行活跃讨论的 CSM，在不了解正在塑造客户对交易认知的替代方案的情况下进入谈判。

续约的扩展维度是 CS 中最持续被错过的机会。续约对话是讨论客户不断变化的需求以及当前合同范围是否仍然适合的自然节点——既可以捕获相对于实际使用量购买不足的账户，也可以识别以证明范围扩展的方式扩展了用例的客户。没有结构化扩展分析进入续约对话的 CSM，将扩展对话留给了偶然性：如果客户提出来就发生，不提出来就不发生。

**COCO 如何解决这一问题**

COCO 帮助 CS 经理建立全面的续约准备材料，将谈判对话从防御性价格讨论转变为证据主导的价值对话——为 CSM 提供以展示价值为立场进行续约和扩展所需的分析、叙事和战略背景。

1. **账户健康摘要生成**：COCO 综合续约期内的完整账户健康图景——健康分数趋势、参与模式、功能采用、支持体验、里程碑完成和关键联系人稳定性——生成识别优势和风险的结构化续约健康摘要。
   - 生成带有基于证据评估的续约期账户健康叙事
   - 识别账户中 3-5 个最重要的健康因素（积极和风险）

2. **投资回报和价值交付文档**：COCO 帮助量化续约期间交付的价值——将产品使用与客户的具体业务成果相连接，计算可归因于产品的时间节省、效率提升、错误减少或收入影响。
   - 构建与客户原始购买目标挂钩的量化投资回报的价值交付摘要
   - 生成适合与客户高管发起人分享的客户语言投资回报叙事

3. **使用趋势分析**：COCO 分析续约期内的产品使用趋势——识别采用是否扩展或收缩、哪些功能显示增长，以及使用模式对客户参与轨迹的暗示。
   - 生成带有对续约对话含义评注的使用趋势报告
   - 识别支持扩展定价的扩展证据（增长的使用量、新用户群体、新用例）

4. **竞争背景评估**：COCO 帮助 CSM 评估与其续约相关的竞争格局——识别最可能在考虑中的竞争对手、对该账户用例最相关的差异化主张，以及如何主动解决竞争比较。
   - 生成针对账户行业和用例量身定制的竞争定位指导
   - 为最可能的竞争替代方案起草竞争异议回应

5. **扩展机会分析**：COCO 识别对每个续约账户最可能相关的扩展机会——评估席位数量缺口、功能采用缺口、新用例机会和组织扩展信号。
   - 生成带有优先建议的扩展机会评估
   - 起草针对每个账户具体增长信号量身定制的扩展对话开场白

6. **续约谈判策略和准备**：COCO 帮助 CSM 制定续约谈判策略——定义他们的初始立场、目标结果、让步框架，以及每个预期谈判场景的具体谈话要点和证据。
   - 生成带有立场、策略和关键谈话要点的续约谈判简报
   - 为续约对话起草开场陈述和投资回报叙事

**可量化的成果**

- **续约率改善**：使用 COCO 准备续约材料的 CS 团队，与没有结构化续约准备的团队相比，续约率高 8-12 个百分点
- **价格让步减少**：基于证据的续约准备通过将对话从价格转向价值，将平均价格让步减少 22%
- **扩展附加率**：在续约对话中使用结构化扩展分析的 CSM 实现 2.4 倍更高的扩展附加率（在续约时签署的扩展交易）
- **续约准备时间**：COCO 辅助的续约材料准备将单次续约准备所需时间从 3-4 小时减少到 45-60 分钟
- **客户对续约流程的满意度**：续约由拥有全面账户健康文档的 CSM 主导的客户，对续约体验的评价比 CSM 以最少准备进入对话的客户高 31%

**受益角色**

- **拥有续约关系的客户成功经理**：需要带着全面的、基于证据的准备进入续约对话，使他们能够从展示价值而非防御性不确定性的立场进行谈判
- **CS 总监和副总裁**：对续约率目标负责，需要在团队中系统化续约准备质量——确保准备质量不依赖于个别 CSM 的经验或尽责性
- **销售和收入运营团队**：在扩展和续约策略上与 CS 协作，需要结构化的账户分析来为每个续约机会的商业策略提供信息
- **客户**：正在做出年度投资决策，从能够清晰阐明已交付价值和持续投资前瞻性案例的 CS 经理处受益

## 实用提示词

**提示词 1 — 完整续约准备材料**
```
我有一个 [30 天] 后到期的战略续约，需要准备全面的续约材料。请帮我建立完整的准备套件。

账户概览：
- 公司：[名称、行业、规模]
- 合同：[$X 年经常性收入，[X] 年合同，续约日期：[日期]]
- 当前联系人：[姓名、职位、关系质量：强健/良好/脆弱]
- 高管发起人：[姓名、职位、参与程度：主动/被动/缺席]
- 关键支持者：[姓名、职位、支持者稳定性：稳定/有风险]

健康和使用数据（续约期内）：
- 总体健康分数趋势：[从 X 到 X，在续约期内]
- 功能采用：[当前采用广度、趋势]
- 产品使用量：[趋势——增长/稳定/下降]
- 支持工单历史：[数量、严重性、值得注意的事件]
- 季度业务回顾出席情况：[多少次，谁参加]
- 合同期间实现的关键里程碑：[列表]
- 合同期间的任何服务事故：[描述]

业务背景：
- 他们购买时声明的业务目标：[列表]
- 实现的业务成果（如已知）：[描述，如有可用则附指标]
- 合同期间的任何组织变化（支持者更换、重组）：[描述]
- 已知的竞争评估：[他们在与竞争对手谈话吗？]
- 预算情况：[您对其预算环境了解什么？]

请生成完整的续约准备材料，包括：
1. 账户健康叙事（2 段）
2. 带有量化成果的价值交付摘要
3. 风险因素和缓解策略
4. 竞争背景和谈话要点
5. 扩展机会评估
6. 推荐谈判策略：初始立场、让步框架、退出点
7. 续约对话的开场陈述
```

**提示词 2 — 面向客户高管的投资回报叙事**
```
我需要在续约讨论中向 [客户名称] 的 [CFO/运营副总裁/CEO] 呈报投资回报案例。这位高管没有参与原始购买决策，需要被说服续约代表对业务的良好投资回报。

产品背景：
- 我们产品的功能：[描述]
- 这个客户如何使用它：[他们的具体用例]
- 他们最初的购买理由：[他们为什么购买——他们正在解决什么问题]

我拥有的使用和成果数据：
- 产品使用量：[描述——例如，"200 个用户，每月处理 5,000 笔交易"]
- 时间节省数据（如有）：[例如，"以前需要 X 小时，现在需要 Y 小时"]
- 错误/返工减少：[如相关]
- 收入影响：[如可测量]
- 成本规避：[如相关——例如，"否则需要雇用 X 名额外员工"]
- 定性成果：[描述——例如，"使他们能够启动 X 举措"]
- 合同成本：[年费]

请：
1. 以高管认为可信的格式计算和呈报量化投资回报
2. 构建适合 CFO 受众的 3-5 要点投资回报摘要
3. 起草我可以在口头演示中使用的 2 段投资回报叙事
4. 识别这些数据中应该锚定续约对话的 1-2 个最强证明点
5. 预测并准备对最可能的高管投资回报挑战的回应："我如何知道这个价值是归因于您的产品而非其他因素？"
```

**提示词 3 — 竞争异议准备**
```
我有一个续约即将到期，我知道客户正在评估 [竞争对手名称]。我需要为竞争比较对话做准备。

我的产品与竞争背景：
- 客户的用例：[描述]
- 他们正在评估的竞争对手：[列表]
- 关于竞争对手宣传内容的了解：[描述他们的定位、定价方式、关键主张]
- 我们针对这个具体用例的关键差异化优势：[列表——要具体]
- 竞争对手更强的领域：[诚实评估]
- 价格定位：[我们是否更贵？客户如何看待价格差距？]
- 客户评估替代方案的主要原因：[是什么促使了竞争评估？]

请准备：
1. 针对此账户的竞争定位简报：对于其特定用例和情况，我们最有说服力的 3-4 个差异化优势是什么？
2. 对于每个预期的竞争异议，一个回应框架：承认 → 差异化 → 证据
3. 对价格差距异议的回应（如果我们更贵）：如何将价格重新框架为投资回报而非成本
4. "为什么是现在"的回应：如果客户想推迟决定，延迟的成本是什么？
5. 我应该问哪些问题以更好地了解竞争评估，以及他们留下来需要哪些条件
```

**提示词 4 — 续约中的扩展对话**
```
我即将与 [客户名称] 进行续约对话，我想利用这个时机推进扩展对话。我已经识别了潜在的扩展机会，但需要准备方法。

当前合同：
- 产品：[他们拥有什么]
- 年经常性收入：[$X]
- 席位/使用量：[当前]

我识别的扩展机会：
1. [机会 1：例如，"他们在专业层级有 50 个席位，但有 150 名员工——显著的席位扩展机会"]
2. [机会 2：例如，"他们在使用我们的分析模块但不使用我们的自动化模块，后者解决了我在上次季度业务回顾中听到的工作流痛点"]
3. [机会 3：例如，"他们 6 个月前收购了一家公司，被收购公司正在使用竞争对手——整合机会"]

对于每个扩展机会，请：
1. 估算年经常性收入机会
2. 识别使这个对话在续约背景下自然发生的最强触发点/钩子
3. 识别最能从扩展中受益且应该作为内部支持者的利益相关者
4. 起草 2-3 句自然地从续约健康对话过渡到扩展机会的对话开场白
5. 预测并准备对每个扩展提案最可能的异议

格式化为我在会议前可以回顾的结构化扩展准备简报。
```

**提示词 5 — 续约谈判策略简报**
```
我需要为一个预计会很困难的续约制定谈判策略。客户面临风险，收到了竞争性报价，并可能要求价格降低。

背景：
- 账户：[名称、年经常性收入、层级]
- 续约日期：[X 天后]
- 健康分数：[分数和趋势]
- 竞争情况：[描述竞争对手报价或已知评估]
- 客户声明的关切：[他们对续约说了什么]
- 我们交付的价值：[关键指标——要具体]
- 我们的内部约束：[您有什么折扣权限？可接受的底价是多少？]
- 关系质量：[您与决策者的关系有多牢固？]
- 他们的替代方案：[现实地看，他们能够切换吗？对他们来说切换会有多痛苦？]

请生成谈判策略简报：
1. 情况评估：我们的立场有多强，诚实地说？
2. 初始立场：我应该首先提议什么，为什么？
3. 让步框架：如果我必须在价格上让步，我应该要求什么作为回报？（多年承诺、扩展承诺、案例研究参与、参考协议）
4. 价值锚点：在价格讨论开始之前，我应该首先引用哪 2-3 条具体证据来锚定价值对话？
5. 退出分析：在什么条件下我应该接受重大折扣，vs. 升级，vs. 退出？
6. 起草我将用来在任何定价讨论开始之前框架续约对话的 3 句开场白
```

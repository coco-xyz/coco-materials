# 注意事项

使用 COCO AI 员工时的重要提示和常见机制说明。

## 初始化配置（首次必做）

将 AI 员工添加到群聊后，**第一步不是直接提问，而是先完成以下初始化设置：**

### 第1步：开启群聊白名单

在群内 @AI员工 发送：

```
@AI员工 开启本群白名单
```

AI 员工会确认本群已启用，之后群内成员即可正常交互。

<br>

### 第2步：为需要私聊的成员开通权限（可选）

如果希望某位成员能与 AI 员工 1对1 私聊，在群内发送：

```
@AI员工 开放 @某成员 的私聊白名单
```

> **说明：** 群聊中 @AI员工 交互不受白名单限制，所有群成员均可使用。白名单仅控制私聊权限。

<br>

### 第3步：选择群聊模式（可选）

| 模式 | 触发条件 | 适合场景 | 开启方式 |
|------|---------|---------|---------|
| **标准模式**（默认） | 必须 @AI员工 才回复 | 大群、日常讨论 | 默认已启用，无需操作 |
| **智能群聊模式** | AI 自动判断是否需要介入 | 小型项目群（10人以下） | @AI员工 发送"开启智能群聊模式" |

如需开启智能群聊模式：

```
@AI员工 开启智能群聊模式
```

如需关闭：

```
@AI员工 关闭智能群聊模式
```

> **注意：** 智能群聊模式下 AI 会阅读群内所有消息来判断是否需要回复，Token 消耗会增加。建议仅在核心工作群开启。

---

## 发送第一个任务

初始化完成后，在群聊中 @AI员工 或在私聊中直接发送你的第一个任务：

```
@AI员工 你好！请做一下自我介绍，告诉我你能帮我做什么？
```

AI 员工会根据你的套餐能力给出回复，确认一切正常后即可开始日常使用。

---

## 交互方式速查

| 场景 | 操作方式 | 说明 |
|------|---------|------|
| **群聊中对话** | @AI员工名称 + 消息内容 | 默认模式下，群聊中必须 @ 才会触发回复 |
| **私聊中对话** | 直接发送消息即可 | 需先开通私聊白名单权限 |
| **多人群聊** | @AI员工 后所有群成员可见回复 | AI 的回复对群内所有人可见 |

> **提示：** 在群聊中未 @AI员工 的消息，AI 不会回复也不会处理，不会干扰正常的团队沟通。

## 长期记忆与 SOP 存储

AI 员工具备长期记忆能力，可以记住重要的工作流程（SOP）、偏好设置和业务规则，在后续对话中持续遵循。

### 如何存储 SOP

直接在对话中告诉 AI 员工你希望它记住的内容：

**示例：**

```
请记住以下工作流程：
1. 每天早上9点检查邮箱中的客户询盘
2. 将询盘内容整理成标准格式：客户名称、需求描述、联系方式
3. 按紧急程度分为A/B/C三级
4. A级当天回复，B级48小时内回复，C级一周内回复
```

```
请记住：我们公司的品牌色是 #F5C542，所有设计稿必须使用这个颜色。
```

```
请记住：回复客户邮件时，结尾统一使用以下签名：
Best regards,
[姓名]
[公司名] | [职位]
```

### 记忆管理

| 操作 | 说明 |
|------|------|
| 添加记忆 | 对话中说「请记住...」或「以后都要...」 |
| 查看记忆 | 问「你记住了哪些规则？」或「列出我之前告诉你的所有SOP」 |
| 修改记忆 | 说「把之前的XX规则改为...」 |
| 删除记忆 | 说「忘掉之前关于XX的规则」 |

> **提示：** 长期记忆在同一个 AI 员工实例内持续有效，即使重新开始对话也会保留。建议将重要的业务规则、工作流程、格式要求等存入长期记忆。

### 适合存入长期记忆的内容

- **工作流程（SOP）**：日常操作步骤、审批流程、汇报格式
- **品牌规范**：品牌色、字体、tone of voice、禁用词
- **业务规则**：定价策略、折扣权限、服务条款
- **沟通偏好**：邮件格式、回复模板、语言风格
- **团队信息**：组织架构、对接人、职责分工

## 其他常见注意事项

### 消息长度限制

| 渠道 | 单条消息限制 | 超长消息处理 |
|------|------------|------------|
| Telegram | 4096 字符 | 自动拆分为多条消息发送 |
| Lark/飞书 | 约 30,000 字符 | 超长内容会自动截断 |

### 文件处理

AI 员工支持处理以下类型的文件（需开通 `im:resource` 权限）：

- **文档**：PDF、Word、Excel、PPT
- **图片**：PNG、JPG、GIF
- **文本**：TXT、CSV、JSON、Markdown

> **注意：** 文件大小建议不超过 20MB。大文件处理可能需要较长时间。

### 响应时间

| 场景 | 预计响应时间 |
|------|------------|
| 简单问答 | 3-10 秒 |
| 文件处理 | 10-30 秒 |
| 复杂分析 | 30-60 秒 |
| 长文档生成 | 1-3 分钟 |

> **提示：** 如果 AI 员工长时间未响应（超过 2 分钟），可以尝试重新发送消息。

### 安全与隐私

- AI 员工不会将你的对话数据分享给其他用户
- 每个 AI 员工实例的数据是隔离的
- 建议不要在对话中发送敏感的密码、密钥等信息
- 企业版支持自定义数据保留策略

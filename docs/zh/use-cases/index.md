# 用例库

探索AI数字员工如何在各行各业、各个岗位上改变工作方式。

## 按类别浏览

### [开发团队](/zh/use-cases/dev-team/001-agent-team-mvp)
AI驱动的开发工作流——更快交付MVP、自动化代码审查、管理技术债务。

### [运营](/zh/use-cases/operations/auto-report-generation)
自动化日报、邮件管理、数据录入和日常运营任务。

### [市场营销](/zh/use-cases/marketing/)
内容创作、社交媒体运营、竞品分析和营销活动自动化。

### [产品](/zh/use-cases/product/)
用户研究、功能优先级排序、路线图管理和产品分析。

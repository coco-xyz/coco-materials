# 企微 — 群成员共用 Bot

## 需求场景

将 COCO AI 机器人拉入企业微信群聊，让群内指定成员都能 @Bot 进行对话和使用。

## 操作步骤

### 第一步：拉 Bot 进企微群

1. 打开目标群聊
2. 点击右上角添加群成员
3. 搜索 Bot 名字并添加

### 第二步：添加可使用成员

1. 在群聊中点击 Bot 头像，选择「去管理」
2. 点击「详情」→「编辑」→「修改」
3. 搜索并添加需要使用 Bot 的群成员

### 第三步：授权权限

1. 点击「可使用权限」
2. 点击「授权」
3. 保存设置

### 第四步：开启群聊白名单

在群内 @Bot，发送：

```
开通本群群聊白名单
```

设置完成后，已授权的群成员 @Bot 即可正常使用。

## 注意事项

- 需要先在企微管理后台添加成员的使用权限，否则 @Bot 不会响应
- 权限修改需要群主或管理员操作

## 简介

在受监管行业（金融服务、医疗、能源、电信、制药）运营的大型企业面临持续涌入的监管文件、指导文件、规则制定通知、执法行动和监管函件。金融服务公司的法律和合规团队可能需要同时监控来自10至15个机构的监管输出。每个机构每月发布数十份文件。监管文件通常体量庞大——一份美国SEC最终规则可能长达500至800页，附带密集的脚注；一份CMS年度支付规则通常超过1000页。

在大多数企业中，当前的监管解读实践是监管监控服务（标记文件但不提供战略分析）、律师事务所提醒（选择性摘要，收费每小时1500至3500元）和内部律师在时间压力下扫描文件的组合。结果是成本高、覆盖不一致、合规意识频频出现盲点。

COCO将监管文件消化转化为结构化、可扩展的情报工作流：

1. **文件摄入**：合规分析师上传监管文件，连同公司的监管背景（哪些业务活动受影响、适用哪些监管机构）。
2. **结构化摘要**：COCO按标准章节生成结构化摘要：变更内容、受影响方、生效/合规日期、必要行动和待解决问题。
3. **影响评估**：COCO将文件与公司当前实践和既往合规文件进行交叉比对，识别新指导要求运营变更的领域。
4. **行动项提取**：COCO识别文件中的每项明确或隐含的合规义务，将其转化为独立行动项，并分配建议的责任人类别。
5. **评论回复支持**：当文件开放公众评论时，COCO分析拟议规则并起草聚焦于对公司业务利益最相关条款的评论信大纲。
6. **监管日历更新**：COCO提取所有规定的截止日期，并将其添加至结构化合规日历，附带提前预警。

使用该工作流的企业报告，主要文件的平均监管文件审查时间从8至12小时缩短至90分钟。合规团队以相同资源水平覆盖3至4倍的监管输出。合规截止日期遗漏（手动流程下的常见风险）降至接近零，因为COCO系统性地提取并记录每个规定的截止日期。

**受益群体：**
- **监管律师和合规律师**：必须监控并分析来自多个机构和司法管辖区的监管输出。
- **首席合规官**：需要在新指导发布当天收到可执行的监管情报，而非两周后。
- **政府事务和公共政策团队**：监控监管趋势并需要参与规则制定评论程序。
- **运营和技术负责人**：必须了解监管文件对其系统和流程的影响，而无需阅读数百页密集监管文本。

## 5个实用提示词

**提示词1 — 监管文件执行摘要**
```
请为[公司名称]（一家受[机构/机构们]监管的[公司类型]）总结以下监管文件。

生成包含以下章节的结构化摘要：
1. 文件概述（类型、发布机构、发布日期、档案编号）
2. 变更内容（新要求与现行规则的对比）
3. 受影响方（哪些实体、活动或交易在范围内）
4. 关键合规日期（生效日期、合规截止日期、评论截止日期（如适用））
5. 我方公司的必要行动（我们必须采取的具体步骤）
6. 需要进一步分析的待解决问题或模糊领域
7. 对我方公司的风险级别：高/中/低

监管文件：
[粘贴文件文本或关键章节]

我方业务背景：
[描述相关业务活动]
```

**提示词2 — 与当前实践的影响评估**
```
将以下新监管要求与我方当前合规计划和实践进行比对。识别：
1. 我方当前实践已满足新要求的领域
2. 我方存在必须解决的合规差距的领域
3. 新要求模糊、需要监管指导或法律意见的领域
4. 每个差距的预估修复工作量：低/中/高
5. 建议的修复优先级：立即/30天/90天/下一周期

新要求：
[粘贴相关规则或指导文本]

我方当前实践：
[描述当前合规计划要素]
```

**提示词3 — 行动项提取和责任人分配**
```
从以下监管文件中提取所有合规义务。对每项义务：
1. 用通俗语言说明具体要求（无监管引用术语）
2. 识别合规截止日期
3. 分配给最适合的责任职能：法务/合规/IT/运营/财务/人力资源/全体
4. 分类为：新要求/变更要求/对现有要求的澄清
5. 标记需要跨职能协调的要求

监管文件：
[粘贴文件文本]
```

**提示词4 — 评论信大纲**
```
我方正在准备对以下拟议规则的评论信。我方公司的利益和关切如下：
- 受影响的主要业务活动：[列出]
- 对拟议规则的主要异议：[列出]
- 我方偏好的替代方法：[列出]
- 我方可提供支持我方立场的数据或证据：[列出]

请起草一份评论信大纲：
1. 以公司简介和我方在规则制定中的利益开头
2. 识别3至5项最令人关切的条款，附具体章节引用
3. 对每项条款，说明我方关切和建议的替代方案
4. 以请求特定监管豁免作为结尾
5. 结构应对审查评论的监管经济学家或政策分析师具有说服力

拟议规则：
[粘贴相关章节]
```

**提示词5 — 执法行动分析**
```
分析以下监管执法行动，提炼适用于我方合规计划的经验教训。识别：
1. 触发执法行动的行为
2. 援引的具体监管条款
3. 执法行动揭示的系统性合规失败（不仅是具体事实）
4. 我方当前合规计划是否解决了导致此次执法行动的问题
5. 我方应考虑对计划或政策进行的具体修改
6. 是否应向我方董事会或审计委员会汇报此次执法行动及原因

执法行动：
[粘贴执法行动或摘要]

我方公司背景：
[描述相关业务活动和合规计划]
```

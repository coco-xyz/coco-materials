# 钉钉 — 群聊中使用 Bot

## 需求场景

将 COCO AI 机器人拉入钉钉群聊，让群内所有成员都能 @Bot 进行对话和使用。

## 操作步骤

### 第一步：发起群聊

在钉钉中创建一个新群聊（或使用已有群聊）。

### 第二步：添加机器人

在群设置中选择「添加机器人」，将你的 COCO AI Bot 拉入群聊。

### 第三步：开启群聊白名单

在群内 @Bot，发送：

```
开启本群群聊白名单
```

设置完成后，其他群成员 @Bot 即可正常使用。

## 注意事项

- 如果其他成员 @Bot 后没有回复，群主可以直接在群里问 Bot：「别人 @你为什么不回」，Bot 会自动检查并修复配置

## 简介

保密协议（NDA）是商业中谈判最频繁的合同。一家中型企业每年可能签署200至500份NDA——涵盖潜在合作关系、供应商评估、并购谈判、客户试点和员工离职。尽管NDA无处不在，其谈判却消耗了内部法务团队不成比例的大量精力。公司法律顾问协会的调查显示，NDA审查和谈判是内部法务团队最耗时的三项工作之一，但大多数法务负责人认为这是一项低价值活动，分散了对战略性工作的注意力。

NDA谈判中的核心矛盾有据可查：提出其标准NDA的一方将其起草为有利于己方的版本（更宽泛的保密信息定义、单向义务、永久期限、宽泛的不招揽条款）；接收方则希望限制范围、确保双向性、限制期限并排除既有知识。这是一种已知的、有规律的谈判——几乎每份NDA都会出现相同的15至20个问题。然而法务团队却在每次谈判中从零开始处理，在本质上只是模式匹配的工作上花费每份45至90分钟。

COCO将NDA修订从劳动密集型审查转变为快速、符合政策的工作流：

1. **NDA上传和当事方识别**：律师上传对方的NDA，并说明各方角色（披露方、接收方、双向）和商业背景（合作讨论、并购尽职调查、供应商评估）。
2. **政策应用**：COCO应用公司的标准NDA谈判手册——保密信息的首选定义、必要排除、最长期限、必要的返还/销毁条款、首选适用法律和任何不可谈判立场。
3. **问题识别**：COCO识别偏离公司政策或市场规范的每个条款，并对每个偏差进行风险评估。
4. **修订生成**：COCO使用公司首选语言生成完整的NDA修订版本，使用标准法律标注约定（删除内容用删除线，新增内容用下划线）。
5. **谈判说明**：COCO为每项修改生成简短谈判备忘录，解释修订原因以及若对方反驳公司将接受的最低可接受替代方案。
6. **上报标记**：COCO识别需要在发送修订版前经过高级律师或业务决策者审查的条款。

采用该工作流的组织报告，NDA处理时间从2至3个工作日缩短至4至6小时。每年处理400份NDA的法务团队每年估计节省300至600个律师小时。销售团队报告更高的交易速度，因为法律审查不再在销售漏斗顶部造成瓶颈。NDA风险一致性——通过已接受的NDA是否包含已知风险条款来衡量——显著提高，因为政策以机械方式而非靠记忆应用。

**受益群体：**
- **内部律师和助理律师**：处理大量NDA审查，需要在不牺牲政策合规的情况下更快处理更多协议。
- **业务拓展和销售负责人**：需要更快的法律审批以推进合作伙伴和客户对话。
- **法务运营管理人员**：跟踪NDA周期时间，寻求将执行时间作为法务效率指标来缩短。
- **并购和企业发展团队**：需要在活跃的交易流程中快速、一致地处理目标方NDA。

## 5个实用提示词

**提示词1 — 完整NDA修订**
```
请从[公司名称]的角度审查并修订以下NDA。我方身份为[披露方/接收方/双方（双向）]。

商业背景：[描述：如供应商评估、合作讨论、并购尽职调查]

我方标准立场：
- 保密信息定义：[首选范围]
- 必要排除：[如：已公开、独立开发、无限制地从第三方处获得]
- 期限：[首选期限，如：披露后2年]
- 终止时的义务：[在X天内返还或销毁]
- 适用法律：[司法管辖区]
- 不招揽：[接受/拒绝/首选范围]
- 允许向员工/顾问披露：[首选语言]

对每项修改：
- 说明修改内容和原因
- 使用标准修订标注（删除线表示删除，下划线表示新增）
- 注明这是"必须有"还是"首选但可谈判"的修改

NDA文本：
[在此粘贴NDA]
```

**提示词2 — NDA问题识别**
```
请审查以下NDA，识别所有以下条款：
1. 超出[双向/单向]NDA市场标准的义务
2. 基于我方业务模式（[描述业务]）可能给我方造成意外义务的条款
3. 可能被解释为不利于我方的模糊语言
4. 与我方任何现有法律义务相冲突的条款（如：上市公司披露要求、监管报告）
5. 签署前应标记的不寻常或非标准条款

对每个问题，请提供：章节引用、问题语言、风险和建议修复方案。

NDA：
[在此粘贴NDA]
```

**提示词3 — 对方反驳应对**
```
我方向[对方]发送了修订版NDA。对方对我方请求的以下修改提出异议。对每项异议，请建议：
1. 其反提案是否在市场规范范围内
2. 我方应接受、反提案还是坚守立场
3. 如需反提案，请起草反提案语言
4. 接受对方立场与我方立场相比的商业风险
5. 该问题是否值得上报给业务决策者

我方原始修订：
[描述我方请求的修改]

对方回应：
[粘贴其反提案或描述其异议]
```

**提示词4 — NDA期限和范围分析**
```
专门针对范围和期限风险分析以下NDA条款：

1. "保密信息"定义——对于[商业背景]而言，范围是否过宽、过窄或适当？
2. 排除条款——是否存在标准排除（公共领域、独立开发、第三方披露、法律要求）并且起草恰当？
3. 期限——保密义务持续多长时间，对于共享的信息类型是否适当？
4. 剩余信息条款——是否存在削弱保密义务的剩余信息条款？
5. 竞业禁止或不招揽条款——这些条款是否超出声明目的所需的范围？

NDA：
[在此粘贴NDA]
```

**提示词5 — NDA政策合规检查**
```
由于[原因：如对方是大型企业不接受谈判/时间压力]，我们正在评估是否按原样签署以下NDA而不进行谈判。

请通过以下检查评估按原样签署该NDA是否在我方可接受风险范围内：
1. 我方任何不可谈判的政策立场是否被违反？[列出不可谈判项]
2. 如果按原样签署，最坏情况的法律风险是什么？
3. 是否存在任何可能干扰我方现有业务运营的条款？
4. 建议决定：按原样签署/在了解风险的情况下签署/不签署

NDA：
[在此粘贴NDA]
```

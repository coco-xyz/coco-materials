# 飞书/Lark — 群成员共用 Bot

## 需求场景

将 COCO AI 机器人拉入飞书/Lark 群聊后，让群内所有成员都能直接使用 Bot，无需每人单独部署。

## 操作步骤

### 第一步：创建群组

在飞书/Lark 中创建一个新群组（或使用已有群组）。

### 第二步：将机器人拉进群

在群设置中添加机器人，搜索并选择你的 COCO AI Bot。

### 第三步：开启群聊白名单

在群内 @机器人，发送：

```
开启本群群聊白名单
```

设置完成后，群内其他成员即可直接 @机器人 进行对话和使用。

## 注意事项

- 受飞书/Lark 平台限制，**暂不支持将机器人拉入外部群**并使用
- 群成员需要 @机器人 才能触发回复（除非单独配置了「接收所有消息」模式）

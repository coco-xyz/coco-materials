## 简介

绩效评估是员工生命周期中最重要——也最令人望而生畏——的流程之一。它塑造薪酬决策、晋升路径、发展规划，并最终影响员工留存。做得好，它能为员工提供清晰、可落地的反馈，加速成长并强化高绩效；做得差，它会侵蚀信任、引发法律风险、造成薪酬不公，并无法有效区分高绩效者与低绩效者。

关于绩效评估质量的研究结论发人深省。盖洛普 2019 年的研究发现，仅有 14% 的员工强烈认为绩效评估激励了他们去改进。企业执行委员会的数据显示，95% 的管理者对公司的绩效管理流程感到不满。德勤的全球人力资本趋势调查揭示，58% 的企业认为当前的绩效管理方式既无法提升员工敬业度，也无法驱动高绩效。

核心问题在于执行，而非意图。大多数组织都有合理的绩效框架——明确的评级量表、胜任力模型、OKR 或 SMART 目标等目标设定方法。真正崩溃的地方，是将这些框架转化为实际书面评估的过程。管理者通常是其职能领域的专家，而非专业写手或反馈教练，却要在季度或年度截止日期前，对每一位团队成员产出书面评估。结果可想而知参差不齐：部分评估详细、具体、以发展为导向；许多则流于泛泛、措辞模糊，或短得无法提供任何可落地的指导。

后果层层叠加。模糊的评估使以绩效为依据的薪酬决策在受到质疑时无从举证。不一致的评级量表引入偏见——研究表明，在非结构化流程中，性别、种族和外貌会系统性地影响绩效评级。读起来像套话的评估既无法激励高绩效者，也无法向低绩效者清晰传达问题所在，让两类员工都感到被忽视。

COCO 通过成为管理者的智能起草伙伴，从根本上改变了评估撰写流程。管理者无需面对空白文本框发愁，只需向 COCO 提供关键绩效数据——目标、成果、发展领域、具体事例——COCO 即可起草一份完整、均衡、基于行为的评估，与公司的评估框架和语调标准高度契合。

**COCO 的解决步骤：**

1. **管理者信息收集**：管理者向 COCO 提供员工姓名和职位、该周期的绩效目标、附有支撑事例的关键成果、观察到的发展领域，以及整体评级。几个要点即可——撰写由 COCO 完成。

2. **结构化草稿生成**：COCO 生成包含所有标准模块的完整评估：整体总结、对照目标的成果（附每项的具体证据）、核心胜任力的体现、发展领域（以建设性方式表述），以及前瞻性目标或发展承诺。

3. **语调校准**：COCO 根据员工评级校准语言——高绩效者的评估强调影响力和拓展空间；平均绩效者的评估在认可与明确发展方向之间保持平衡；低绩效者的评估直接、具体、文档完整，但不带惩罚性。

4. **偏见审计**：COCO 审查草稿中与绩效评估偏见相关的语言模式——对女性的模糊褒奖与对男性的决断性成就描述、人格导向的反馈与行为导向的反馈、努力导向的表扬与影响导向的表扬——并在评估定稿前标记或纠正这些模式。

5. **框架对齐**：HR 团队将组织的胜任力框架和评级定义配置到 COCO 中，确保所有管理者起草的评估都与同一标准一致校准。

6. **评审周期效率**：对于团队规模较大的管理者，COCO 大幅压缩了评估撰写时间——让他们能够为每位团队成员产出深思熟虑、个性化的评估，而不是在最后几份评估时质量急剧下滑。

**可量化成果：**

- 管理者花在绩效评估撰写上的时间平均减少 65%
- 评估质量评分（由 HR 校准团队衡量）提升 48%
- 三分之一的初始管理者草稿中检测到偏见标记——经 COCO 修正后，偏见性措辞减少 72%
- 员工对绩效反馈质量的满意度提升 39%
- 由于语言更加具体、基于行为，与绩效评估文档相关的法律质疑减少 60%

**受益角色：**

- **管理者和团队负责人**：需要为多位直接下属撰写评估，苦于时间投入和产出真正有用的评估所需的写作质量
- **HR 业务伙伴**：花大量时间辅导管理者提升评估写作质量、校准不一致的评级、清理法律风险措辞
- **HR 运营团队**：负责管理评审周期，需要在整个组织内产出一致、结构化的评估
- **员工**：获得更具体、可落地、公平的反馈——实现真正的发展对话，而非每年一次的走过场

## 实用提示词

**提示词 1 — 高绩效员工评估**
```
请为 [Sarah Kim]（[高级产品设计师]）撰写 [2024 年第四季度 / 2024 年度] 评审周期的绩效评估。评级：[超预期]。
主要成果：
- [主导核心结账流程重新设计，将放弃率降低 23%]
- [辅导 2 名初级设计师，两人均获得积极的同事反馈]
- [尽管团队人员削减，所有重大项目均按时交付]
发展领域：[可在跨部门沟通方面投入更多——利益相关方有时感到信息不够及时]。
需要突出的胜任力：[卓越设计、协作能力、影响力导向]。
语调：[温暖、具体、面向未来]。评估字数：[400-500 词]。
```

**提示词 2 — 需要改进且需留存文档的评估**
```
请为 [Tom Bradley]（[区域销售经理]）起草 [2024 年度评审] 绩效评估。评级：[需要改进 / 接近绩效改进计划（PIP）]。
绩效背景：[Tom 在第二、三、四季度均未达到配额。销售运营团队持续标记其管道管理问题。他的团队今年离职率达 40%，显著高于公司平均水平 18%]。
主要问题：[管道管理、团队领导力、对结果的责任感]。
需认可的优势：[扎实的产品知识、在企业大客户中建立了良好的客户关系]。
要求语调：[直接、具体且具法律可辩护性——聚焦于行为和结果而非性格。必须包含具体事例，不得泛泛而谈。包含一个前瞻性部分，明确下一评审周期的清晰期望]。
```

**提示词 3 — 员工自我评估草稿**
```
帮我撰写在 [金融科技公司] 担任 [高级数据分析师] 的 [年度绩效评审] 自我评估草稿。评审周期：[2024 年]。
主要成就：[构建了自动化收入对账仪表板，每周节省 15 小时的手动工作量；主导将遗留报表从 Excel 迁移至 Tableau；首次向管理层展示季度业务回顾分析]。
我希望发展的领域：[高管沟通、大数据集的 SQL 查询优化]。
我希望语调 [自信但不自大，聚焦结果，专业规范]。字数：[350-400 词]。
```

**提示词 4 — 现有草稿偏见审查**
```
请审查这份绩效评估草稿是否存在潜在偏见，并提出改进建议。被评估员工是 [一位在工程岗位的女性]。草稿内容如下：[粘贴草稿]。请重点检查：(1) 使用了本应对男性同行使用主动性语言之处反而使用了共情性语言（"她非常支持他人"vs."他推动了结果"）；(2) 努力表扬而非影响表扬（"她真的很努力"vs."她交付了 X 成果"）；(3) 不会出现在男性同行评估中的性格批评；(4) 削弱成就描述的模糊限定词。请提供修订版本。
```

**提示词 5 — HR 校准会议摘要**
```
我正在准备 [第四季度评审校准会议]。我有 [12] 位直线管理者的评估需要呈现。请帮我创建一个校准汇总表，列出每位员工、管理者建议的评级、支撑该评级的 2-3 个关键证据点，以及若评级与所提供证据存在明显不一致时的标记。以下是所有评估：[粘贴 12 份评估摘要]。
```

# 用例 #282：AI 高管沟通起草助手

**角色**：高管 | **行业**：企业 | **任务**：邮件管理、高管沟通

---
## 详细介绍

**痛点：高风险的高管沟通需要数小时的精准打磨——而领导者根本没有这些时间**

高管沟通所适用的标准与普通商业写作截然不同。关于组织重构的全员信、季度目标未达成后的投资者更新、请求治理变更的董事会备忘录，或是领导层变动的通知——每一份这样的文件都可能在数月内塑造组织信任、投资者信心或市场认知。风险之高，以至于哪怕一个措辞不当的短语、一个对焦虑情绪的忽视、或一段语气失调的段落，都可能成为舆论焦点，将真正的内容完全遮蔽。高管深知这一点，这正是他们在这些沟通上反复斟酌的原因——即便时间是他们最稀缺的资源，仍然常常在一条关键消息上花费3至5小时。

挑战不在于写作能力。大多数高管在对话中都是有效的沟通者。挑战在于：在约束条件下追求精准——需要在一份将被数十乃至数千名具有截然不同背景与关切的利益相关者阅读的文件中，同时管理好语气、结构、受众预期、法律敏感性、情感共鸣和战略定位。董事会备忘录必须具备分析性和事实依据；全员邮件必须体现人情味和安抚感，而不显得居高临下；给投资者的信必须传递信心，同时诚实地应对挑战。在这些不同的表达模式之间切换，需要一种刻意的写作技艺，而这在初稿中并不会自然涌现，尤其是在时间压力下。

结构性的时间消耗来自迭代环节。高管写完初稿，意识到不到位，进行修改，传给公关顾问或法务团队，整合反馈，再修改，拿给CEO或董事会主席审阅，第三次修改，最终发出。即便这些沟通材料需要在数小时内完成，这一流程也往往需要数天时间。流程还因各方审阅者对语气的不同意见而进一步复杂化，在不丢失高管本人真实声音的前提下调和这些意见，本身就是一项不容小觑的编辑挑战。最终产出的沟通内容往往安全却缺乏生命力——风险被清除了，但信念也一并消失了。

敏感沟通还带有额外的复杂性：如何组织坏消息的框架。无论是传达裁员、目标未达成、领导层变动还是市场挑战，高管都必须在诚实与稳定之间寻求平衡——既承认现实而不引发恐慌，既保持问责感而不让他人感到被指责，又需要展示可信的前进路径而不做出组织无法兑现的承诺。要在这之间找到恰当的平衡点极为困难，需要同时理解受众的情绪状态、组织的信任基础，以及在法律层面哪些可以披露、哪些不能。

**COCO 如何解决这一问题**

COCO 通过生成符合高管声音、受众需求和沟通目标的精磨初稿，加速高管沟通起草流程——将迭代周期从数天压缩至数分钟。

1. **声音与语气校准**：COCO 调整输出内容以匹配特定高管的沟通风格及受众定位。
   - 分析样本沟通内容，复制高管的自然语言表达模式
   - 在董事会（分析性）、全员（人性化）、投资者（自信）和媒体（可控）之间适当切换表达档位
   - 避免使真实高管声音趋于平淡的企业套话

2. **受众特定的结构框架**：COCO 根据特定受众的信息需求和情绪状态来组织每份沟通内容。
   - 董事会沟通：问题陈述、证据、选项、建议、风险分析
   - 全员信息：对受众现实处境的确认、清晰的叙事、"这对你意味着什么"、下一步行动
   - 投资者更新：背景、业绩回顾、前瞻性叙事、问答准备
   - 对外媒体声明：以事实开篇，提供背景，表明回应，表示可接受进一步跟进

3. **敏感消息框架**：COCO 运用经过验证的框架传递困难沟通内容，避免引发不成比例的反应。
   - 在陈述事实之前先传递共情与确认
   - 以背景和前瞻性承诺为坏消息提供叙事框架
   - 避免发出规避信号的被动语态表述
   - 识别需要标注请法律顾问审查的法律敏感措辞

4. **多版本变体生成**：COCO 生成针对不同语气和篇幅要求的多个版本。
   - "果断自信"与"共情审慎"的语气变体
   - 长篇详细版本与简短执行摘要版本
   - 适用于不同渠道背景的正式和非正式风格变体

5. **一致性与对齐检查**：COCO 审查草稿沟通内容与以往声明的一致性。
   - 标注与先前董事会陈述或投资者沟通相矛盾的措辞
   - 确保多渠道沟通推进中的关键信息保持一致
   - 识别草稿语言中无意嵌入的承诺或允诺

6. **下游沟通级联**：COCO 起草重大公告所需的全套沟通材料。
   - 主要高管信息 + 经理传达要点
   - 面向内外部受众的FAQ文件
   - 针对不同利益相关者群体的后续沟通
   - 在紧急情况下正式全文准备期间使用的临时声明

**可量化的成果**

- **从起草到发送的周期时间**：以COCO结构化初稿为起点，平均高管沟通周期从3至5小时缩短至45至90分钟
- **修改轮次**：使用AI辅助起草的沟通团队报告在最终批准前的修改轮次减少60%
- **声音一致性**：基于共享AI基线起草的多利益相关者沟通活动，与临时起草相比实现显著更高的信息一致性
- **危机沟通速度**：高管危机响应沟通在20分钟内起草完毕并准备好供审查，而行业平均水平为4小时以上
- **沟通信心**：高管报告对AI辅助起草并经修改完善的沟通内容发出的信心，比完全在压力下独立起草的内容高出40%

**受益角色**

- **CEO及C级高管**：在不牺牲这些关键时刻所需质量与真实性的前提下，降低高风险沟通的认知和时间负担
- **企业公关团队**：以AI生成的初稿为结构化基线，让公关专业人员能够专注于内容精磨、策略设计和利益相关者管理
- **投资者关系专业人员**：更快速地起草一致、合规且有说服力的投资者沟通内容，尤其在财报电话会议或重大信息披露等时间敏感情况下
- **董事会秘书和幕僚长**：管理重大组织公告的完整沟通级联，确保所有利益相关者群体同时收到适当且对齐的信息

## 实用提示词

**提示词 1：全员公告起草**
```
请起草一份由我【高管姓名/职位】发给公司【员工人数】名员工的全员沟通内容。

公告内容：【描述需要传达的信息——例如：组织架构调整、新战略、领导层变动、季度业绩欠佳、公司里程碑】

需要包含的关键事实：
- 【事实1】
- 【事实2】
- 【事实3】

员工目前可能的情绪状态：【例如：对工作安全感到焦虑 / 对新方向感到期待 / 因近期领导层变动而感到不确定】

阅读后我希望他们的感受：【例如：被理解、对领导层有信心、清楚接下来会发生什么】

不应该说的内容：【法律敏感、尚未确定或更适合后续跟进的话题】

我的沟通风格：【例如：直接、不拖泥带水 / 温暖、平易近人 / 正式、结构化】

请起草：
1. 主要全员信息（400至600字）
2. 邮件主题行选项（3个变体）
3. 供经理传达的核心信息摘要（3条要点）
4. 员工可能提出的3至5个常见问题及建议回应
5. 一周后使用的简短跟进信息模板
```

**提示词 2：投资者或董事会沟通起草**
```
请起草一份面向以下对象的【投资者更新 / 董事会备忘录 / 季度信函】：【投资者 / 董事会成员 / 有限合伙人】

背景：
- 公司阶段：【阶段——例如：B轮SaaS / 上市公司 / PE支持的企业】
- 本次沟通触发因素：【是什么促成了本次沟通——例如：季度更新 / 目标未达成 / 战略转型 / 寻求某项审批】
- 需要报告的关键指标：
  - 【指标】：【实际值 vs. 目标值或上一期数据】
  - 【指标】：【实际值 vs. 目标值或上一期数据】
  - 【指标】：【实际值 vs. 目标值或上一期数据】

进展顺利的方面：【列出2至3项积极进展】
未按计划进行的方面：【诚实列出挑战】
前瞻性叙事：【计划是什么 / 为什么对方应该对管理层保有信心】

需要谨慎处理的敏感领域：【需要仔细表述的话题】
目标语气：【自信且直接 / 透明且负责 / 审慎乐观】

请起草：
1. 完整沟通内容（根据受众和目的确定适当长度）
2. 执行摘要段落（5至7句话）
3. 阅读后对方可能提出的5个问题及草拟回应
4. 任何我应该提交法务/合规审查的措辞
```

**提示词 3：领导层变动公告**
```
我需要公告一项领导层变动。请帮我起草相关沟通内容。

变动情况：【描述——例如：【姓名】【职位】离职 / 【姓名】晋升至【职位】/ 【职能】架构调整】
情况说明：【例如：自愿离职寻求新机遇 / 绩效原因 / 退休 / 战略重组】
可以公开说明的内容：【具体说明——哪些已获准披露】
不能说的内容：【法律或敏感性限制】
离任领导与组织的关系：【例如：任期长且深受喜爱 / 近期入职未能奏效 / 存在争议的离职】
过渡安排：【寻找继任者/永久方案期间的安排是什么】

受众：
- 内部（员工）：【内部受众的特殊敏感点】
- 外部（客户、合作伙伴）：【是否需要对外沟通】
- 董事会：【是否需要单独的董事会沟通】

请起草：
1. 内部全员公告
2. 对外公告（如适用）
3. 致离任领导所在团队的直接沟通
4. 经理用于处理后续问题的谈话要点
5. 用于回应外部询问的官方一行声明
```

**提示词 4：危机或敏感事件响应沟通**
```
我需要就一个已经出现的敏感情况进行沟通，需要谨慎的框架处理。

情况描述：【描述问题——例如：数据泄露 / 产品故障 / 公众争议 / 监管行动 / 员工事件】
当前状态：【我们已经知道什么 / 哪些仍在调查中】
已传达的内容（如有）：【之前的声明】
本次沟通的受众：【员工 / 客户 / 公众 / 特定利益相关者群体】
时间线：【紧迫程度如何 / 什么时候需要发出】

约束条件：
- 法律边界：【不能说什么】
- 尚不清楚但与此相关的内容：【已知的未知】
- 我们能做出的承诺：【具体的承诺 / 下一步行动】
- 暂时不能做出的承诺：【避免这些内容】

所需语气：【例如：共情且负责 / 冷静且基于事实 / 紧迫且行动导向】

请起草：
1. 主要响应沟通内容
2. 正式全文准备期间立即使用的临时声明
3. 致员工的内部沟通（与对外沟通分开）
4. 供应对利益相关者询问的一线团队使用的FAQ
5. 有更多信息可披露时的后续沟通模板
```

**提示词 5：战略转型或愿景重塑沟通**
```
我需要就重大战略变化或组织方向的重置进行沟通。

变化内容：【描述战略转变——例如：退出某一市场 / 产品重心转移 / 重大重组 / 全新使命框架】
变化原因：【是什么推动了这一决定——市场状况、数据、董事会方向、业绩表现】
保持不变的部分：【需要强调的重要延续性】
变化对不同群体意味着什么：
- 员工：【对职位、职能、文化的影响】
- 客户：【对产品、服务、合作关系的影响】
- 投资者：【财务和战略含义】

我希望讲述的叙事：【什么是有说服力的前景故事】
沟通不当的风险：【可能出现什么问题——人才流失、客户流失、投资者担忧】

请起草：
1. CEO叙事备忘录（内部）——完整的战略理由
2. 全员版本（简洁、贴近人心、行动导向）
3. 客户沟通内容（如适用）
4. 对外媒体声明或投资者沟通
5. 关键信息框架：在每个版本中必须传递的3条核心信息
6. 常见异议及各项应对方式
```

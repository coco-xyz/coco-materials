# Use Case #012: AI Ad Copy Generator

**Role**: Performance Marketer / Paid Media Specialist / Growth Manager | **Industry**: SaaS, E-commerce, D2C, Fintech, B2B | **Task**: Ad Copywriting, A/B Testing Variations, Multi-Platform Ad Creation, Creative Optimization

---

## 社媒文案

我们的Google Ads账户有340个广告组。每个至少需要3个自适应搜索广告变体。那就是1020条广告文案要写、要测试、要优化。我们的效果营销团队：2个人。

他们60%的时间在写广告文案。剩下的40%？分析哪些文案有效。根本没时间做出价策略、受众测试或落地页优化——你知道的，那些真正影响ROAS的事情。

最糟糕的是：他们手写的广告中70%被Google的Ad Strength评为"差"或"一般"。不是因为他们写得差——是因为他们写到精疲力竭。

COCO的AI Ad Copy Generator把瓶颈变成了流水线：
- 几分钟内为每个广告组生成15个标题+描述变体
- 理解平台特定要求（Google RSA、Meta主文本、LinkedIn赞助内容）
- 使用经过验证的文案框架（PAS、AIDA、利益优先、紧迫感驱动）
- 从我们的获胜广告中学习——新变体继承历史转化的模式
- A/B测试速度从每月2个变体提升到每广告组每月12个变体

3个月后：Ad Strength评分从48%"好/优秀"升至87%。CPC下降22%。CTR提升31%。团队终于有时间优化真正重要的事情——ROAS跳增40%。

## 详细介绍

**痛点：广告文案的量产难题**

效果营销的成败取决于迭代速度。测试更多变体、学习更快、优化更激进的团队获胜。但现代付费媒体要求压倒性数量的创意文案。仅Google的自适应搜索广告就需要每个广告组15个标题和4个描述。Meta建议每个广告集3-5个广告创意变体。LinkedIn、TikTok和其他平台各有自己的要求。

对于一个200+广告组的中型账户，这意味着数千个独特的广告文案变体——所有这些都需要符合品牌、有吸引力、符合平台政策，并且差异化到足以测试出有意义的东西。

大多数效果营销团队的瓶颈不是预算或策略，而是生产文案的物理容量。写手倦怠。质量下降。测试速度放缓。最大的成本不是写作时间——而是测试不够快的机会成本。

**COCO如何解决**

COCO的AI Ad Copy Generator专为效果营销构建，理解各平台付费广告的约束、心理学和最佳实践。

1. **平台原生生成**：COCO理解每个平台的广告格式要求：
   - **Google RSA**：15个标题（各30字符）、4个描述（各90字符）、固定位策略
   - **Meta/Facebook**：主文本（可见125字符）、标题、描述、CTA按钮对齐
   - **LinkedIn**：赞助内容（150字符引言）、InMail主题行、轮播卡文案
   - **TikTok**：短视频脚本、文字叠层、CTA整合
   - **Microsoft Ads**：类似Google但有受众人口统计调整

2. **文案框架智能**：每条广告使用经过验证的框架生成：
   - **PAS**（问题-激化-方案）：以痛点开头，放大它，呈现解决方案
   - **AIDA**（注意-兴趣-欲望-行动）：顺序参与漏斗
   - **利益优先**：以结果而非功能开头
   - **社会证明**：整合数据、评价、信任信号
   - **紧迫/稀缺**：限时优惠、限量供应

3. **基于效果的学习**：COCO分析你的历史广告效果数据：
   - 哪些标题CTR最高？
   - 哪些描述驱动最多转化？
   - 什么情感角度对你的受众有效？
   - 哪些CTA表现最好？
   - 新变体在继承获胜模式的同时测试新角度

4. **批量生成且差异化**：为同一广告组生成多个变体时，COCO确保每个变体测试不同角度：
   - 变体1：利益导向
   - 变体2：痛点导向
   - 变体3：社会证明导向
   - 变体4：紧迫感导向
   - 变体5：问题导向
   确保A/B测试产出有意义的学习，而非微小差异的改写。

5. **合规与品牌安全**：COCO检查生成的文案是否符合：
   - 平台广告政策（无禁止声明、适当免责声明）
   - 品牌指南（批准用词、禁止语言）
   - 行业法规（医疗、金融服务、法律限制）
   - 竞品商标问题

6. **落地页对齐**：COCO读取你的落地页并确保广告文案：
   - 匹配落地页的核心价值主张
   - 使用一致的术语
   - 设定准确的期望（减少信息错配导致的跳出率）
   - 建议落地页改进以匹配高表现广告角度

**可量化的结果**

- **广告文案生产**：提速15倍（每广告组从4小时降至15分钟）
- **Ad Strength评分**：从48%好/优秀提升至87%
- **CTR提升**：跨账户平均+31%
- **CPC降低**：通过更好的质量分数-22%
- **A/B测试速度**：6倍提升（每广告组每月2到12个变体）
- **ROAS提升**：+40%（来自更快的优化周期）

**受益角色**

- **效果营销人员**：专注策略和优化而非写文案
- **PPC代理商**：扩展客户广告账户而不按比例增加文案人员成本
- **增长团队**：更快测试更多角度，更快找到获胜信息
- **电商品牌**：为数百个SKU生成产品特定的广告文案

## 实用提示词

**提示词 1: Google自适应搜索广告生成**
```
为以下内容生成完整的Google自适应搜索广告：

产品/服务：[描述]
目标关键词：[主关键词]
落地页URL：[URL]
目标受众：[我们定位谁]
核心卖点：[列出3-5个独特卖点]
竞品差异点：[我们有什么不同]
优惠（如有）：[折扣、免费试用等]

生成：
- 15个独特标题（每个30字符以内）
  - 混合：利益导向、包含关键词、CTA驱动、紧迫感、社会证明
  - 建议固定位1的标题
- 4个描述（每个90字符以内）
  - 每个使用不同的文案角度
- 建议的广告附加信息：站点链接（4个）、宣传信息（4个）、结构化摘要

确保标题任意组合仍然通顺。
```

**提示词 2: Meta/Facebook广告创意变体**
```
为Meta/Facebook广告活动创建5个文案变体。

产品/服务：[描述]
活动目标：[品牌认知/考虑/转化]
目标受众：[人口统计、兴趣、痛点]
推广内容：[我们在推什么]
落地页：[URL或描述页面]
品牌调性：[描述语气]

5个变体各使用不同角度：
1. 痛点→方案
2. 社会证明/评价风格
3. 使用前后对比
4. 直接利益+紧迫感
5. 问题引导/好奇心缺口

每个变体需要：
- 主文本（在"查看更多"前的125字符内放核心信息）
- 标题（40字符以内）
- 描述（30字符以内）
- 建议CTA按钮（了解更多/注册/立即购买/获取优惠等）
- 建议的图片/视觉方向
```

**提示词 3: A/B测试假设和文案变体**
```
我们目前[产品/关键词]表现最好的广告是：

标题："[当前标题]"
描述："[当前描述]"
当前指标：CTR [X]%, 转化率 [X]%, CPC ¥[X]

生成4个挑战者变体，每个测试一个具体假设：

变体A - 假设：[例如"情感触发将优于理性利益"]
变体B - 假设：[例如"具体数字将优于模糊声明"]
变体C - 假设：[例如"问句格式将提升CTR"]
变体D - 假设：[例如"社会证明将提升信任和转化"]

每个变体提供：
- 广告文案（标题+描述）
- 相对于对照组具体测试什么
- 预期结果和原因
- 统计显著性所需的最小样本量建议
```

**提示词 4: 电商产品Feed广告文案**
```
为我们的产品Feed广告生成文案模板。这些将用产品数据动态填充。

产品类别：[例如：运动鞋、SaaS工具、家居家具]
品牌定位：[高端/性价比/创新/可持续]
目标受众：[谁在买这些]

创建模板：
1. Google购物补充Feed标题（150字符上限）
   - 模板格式：[品牌] + [产品类型] + [核心特点] + [差异化]
   - 3个模板变体

2. Meta动态产品广告
   - 主文本模板（3个变体）
   - 含{product_name}变量的标题模板
   - 描述模板

3. 再营销广告文案（针对购物车放弃者）
   - 紧迫感变体
   - 利益提醒变体
   - 社会证明变体

使用这些产品属性作为变量：{product_name}、{price}、{discount_percent}、{category}、{key_feature}
```

**提示词 5: 多语言广告本地化**
```
将这些广告文案本地化到[目标市场/语言]。不要只是翻译——要适配当地市场偏好、文化细微差异和平台规范。

原始广告（中文）：
1. 标题："[标题]" | 描述："[描述]"
2. 标题："[标题]" | 描述："[描述]"
3. 标题："[标题]" | 描述："[描述]"

目标语言：[语言]
目标市场：[国家/地区]
平台：[Google/Meta/LinkedIn]
字符限制：标题[X字符]，描述[X字符]

每个本地化版本：
- 适配后的标题和描述
- 标注所做的文化适配（例如：当地更有共鸣的价值主张）
- 标记可能需要目标市场法律审查的声明
- 建议添加的当地信任信号（当地支付方式、当地社会证明等）
```

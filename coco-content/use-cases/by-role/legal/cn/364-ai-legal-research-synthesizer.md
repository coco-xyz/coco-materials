## 简介

法律研究是健全法律建议的基础——也是企业法律实践中最耗时、成本最高且执行最不一致的活动之一。律师事务所的初级律师需要6至12小时研究一个复杂的法律问题并撰写备忘录。内部律师通常没有研究人员，在同样的工作上可能需要花费4至8小时，同时还要管理完整的合同和咨询工作量。对于跨越多个司法管辖区的问题——50个州的产品责任分析、12个国家的数据隐私合规分析、30个地点的劳动法评估——研究负担倍增，以至于大多数内部团队根本无法全面完成。

质量问题与时间问题相互叠加。法律研究质量高度依赖于研究者是否知道应查阅哪些来源、如何识别控制权威、如何综合相互冲突的先例，以及如何区分切题案例与表面相似但事实上可区分的案例。

COCO通过结构化综合工作流解决研究质量、速度和知识保留三个维度的问题：

1. **问题框架**：律师说明法律问题、相关司法管辖区和事实背景。COCO确认研究范围并标记任何门槛性问题。
2. **研究大纲**：COCO生成识别关键法律问题、相关法律体系（法规、法规、判例法、二级资料）和研究方法的结构化研究大纲。
3. **综合**：COCO将适用法律综合为连贯的分析——多数规则、少数立场、巡回法院分歧、新兴趋势、近期立法变化——附上适当的AI研究局限性说明。
4. **应用**：COCO将综合的法律应用于特定事实情况，识别法院如何处理类似事实，以及在请求律师所在司法管辖区可能的结果是什么。
5. **反驳论点**：COCO识别反对客户立场的最有力反驳论点并评估如何应对。
6. **备忘录格式**：COCO以标准法律备忘录格式（提出的问题、简要答复、讨论、结论）生成分析，供律师审查和补充经验证的引用。

使用该工作流的团队每个问题的法律研究时间减少60%至75%。研究质量提高，因为COCO系统性地覆盖所有相关法律问题，而不是只关注最显而易见的来源。知识以结构化、可搜索的备忘录形式保存，而不是电子邮件链。内部法律团队以相同资源水平处理3至4倍的咨询请求，减少积压并提高业务伙伴满意度。

**受益群体：**
- **内部法律顾问**：处理各类法律主题的咨询请求，需要更快地生成准确的分析。
- **总法律顾问**：需要在不等待多天研究周期的情况下快速回应董事会和高管层的法律问题。
- **法务运营负责人**：正在构建机构法律知识库，希望以可扩展的方式记录和组织法律分析。
- **外部律师事务所**：可使用COCO加速助理级研究工作，为客户降低成本并改善交付时间。

## 5个实用提示词

**提示词1 — 法律研究备忘录**
```
就以下问题进行法律研究并生成法律研究备忘录：

法律问题：[说明具体法律问题]

司法管辖区：[州/联邦法院/国家]

事实背景：[描述相关事实]

商业背景：[说明重要性——交易、争议、合规决策]

请以包含以下内容的法律备忘录格式输出：
1. 提出的问题
2. 简要答复（2-3句话）
3. 讨论（适用法规、法规和关键案例分析）
4. 应用于我方事实
5. 结论及风险评估：强/中/弱法律立场
6. 需要进一步研究或验证引用核实的待解决问题
```

**提示词2 — 多司法管辖区比较**
```
研究并比较以下法律问题在指定司法管辖区的情况：

问题：[描述法律问题]

需覆盖的司法管辖区：[列出]

对每个司法管辖区：
1. 说明控制性法律标准
2. 识别关键法规、法规或领先案例
3. 描述法院如何将标准应用于与我方类似的事实
4. 评估法律的清晰度：已确立/不确定/积极诉讼中

然后生成比较摘要表，并根据多司法管辖区格局，对[特定商业活动]提出建议。

我方事实：[描述]
```

**提示词3 — 判例法综合**
```
综合以下法律问题的判例法。我已识别以下可能相关的案例——分析每个案例并综合出统一的法律状态。

法律问题：[说明问题]

待分析案例：[列出带引用或摘要的案例]

综合应包含：
1. 识别每个案例建立的规则或标准
2. 识别案例之间的一致之处和冲突之处
3. 识别多数立场和任何重要的少数立场或巡回法院分歧
4. 提取法院在该领域审查的关键因素
5. 将综合规则应用于我方事实：[描述事实]
```

**提示词4 — 监管解释问题**
```
我们需要对以下监管条款如何适用于我方业务活动的指导。

监管条款：[粘贴文本]

监管机构：[命名机构]

我方情况：[描述该条款如何适用于我方]

研究：
1. 机构的明确解释（指导文件、FAQ、评论回应）
2. 解释该条款的任何法院判决
3. 如有，行业实践或无异议函立场
4. 合理解释的范围以及哪个最具可辩性
5. 是否建议寻求正式机构指导或无异议函
```

**提示词5 — 商业决策法律风险排序**
```
我方正在考虑以下商业行动：[描述拟议行动]。

评估以下领域的法律风险并按可能性和严重程度排序：
1. 合同法风险
2. 监管合规风险（识别哪些监管机构可能关注）
3. 劳动法风险
4. 知识产权风险
5. 隐私和数据保护风险
6. 诉讼风险（来自客户、竞争对手、员工）

对每个风险类别：
- 说明具体风险
- 评估可能性：高/中/低
- 评估潜在严重程度：高（存在性/重大财务影响）/中/低
- 建议缓解步骤
- 标记应由专业律师审查的风险

司法管辖区：[主要司法管辖区]
商业背景：[描述]
```

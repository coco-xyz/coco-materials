# #357 — AI合同条款风险分析器

**角色**: 法务
**行业**: 企业
**任务**: 合规
**Slug**: ai-contract-clause-risk-analyzer

---

## 简介

企业法务团队每季度需要审查数百份商业合同。一家业务活跃的中型企业每年可能处理300至500份合同，每份合同从10页到80页不等。资深律师平均每份合同需要花费3至5小时审查条款层面的语言风险——责任上限、赔偿范围、责任限制例外、自动续约陷阱，以及与公司政策相冲突的适用法律条款。按律师每小时2500至4000元的全成本计算，仅合同审查一项每年就需耗费350万至1750万元的法务成本，这还不包括外部律师费用。

核心问题不在于阅读本身，而在于一致性缺失。不同律师采用不同的风险标准。一位律师标记为高风险的条款，另一位在截止日期压力下可能视而不见。条款库逐渐过时。此前谈判中达成的退让底线被遗忘。新的监管要求（各省消费者保护法修正案、最新个人信息保护法规定、修订后的数据跨境传输机制）未能系统性地应用于新收到的合同。最终结果是风险敞口参差不齐，往往只在纠纷、审计或并购尽职调查时才暴露出来——此时补救成本远超预防成本。

COCO将合同条款分析从手动、依赖主观判断的流程转变为系统化、可审计的工作流。具体操作步骤如下：

1. **上传与解析**：律师上传合同（PDF、Word或纯文本格式），并告知COCO合同类型（主服务协议、SaaS订阅协议、供应商服务协议、保密协议）以及己方身份（买方、卖方、授权方、被授权方）。
2. **政策基准设定**：向COCO提供公司的合同谈判手册——可接受的责任上限倍数、首选适用法律、必要的数据处理条款、禁止的单方修改条款等，以上传文件或结构化提示词的形式输入。
3. **逐条分析**：COCO逐条审读合同，标记与手册的偏差，并为每个被标记项目分配风险级别（严重/高/中/低），附上通俗易懂的风险说明。
4. **修订建议**：针对每个被标记条款，COCO使用公司标准退让语言起草替代方案，可直接粘贴至谈判标注版本。
5. **摘要报告**：COCO生成一页纸的执行摘要，列出五大风险事项、建议谈判优先级，以及根据现行政策构成交易障碍的条款。
6. **审计记录**：完整分析结果连同时间戳、版本号和所用政策基准一并保存，形成可辩护的审查流程记录。

采用该工作流的团队报告，平均合同审查时间从每份4小时缩短至45分钟。风险一致性评分（以律师间条款评级一致率衡量）从约60%提升至90%以上。提交外部律师的事项减少35%至50%，因为内部团队能够更自信地解决问题，无需外部验证。

**受益群体：**
- **内部法律顾问和合同经理**：负责初审，需要在不降低质量的前提下提升审查速度。
- **首席法务官和总法律顾问**：需要对大量合同组合保持一致、可审计的风险态势。
- **采购和财务负责人**：签署合同但缺乏法律培训，需要用商业语言表述的风险摘要。
- **合规官员**：必须确保每份合同在执行前符合数据保护、出口管制和监管要求。

## 5个实用提示词

**提示词1 — 完整合同风险分析**
```
你是一位资深商业律师，正在为[公司名称]（一家[公司类型]，所属[行业]领域）审查合同。我方身份为[买方/卖方/授权方/被授权方]。

请逐条分析以下合同。对每个条款，请说明：
- 条款类型和位置（章节编号）
- 是否偏离市场标准条款或我方政策
- 风险级别：严重/高/中/低
- 通俗易懂的风险说明
- 建议的修订或退让语言

我方主要政策立场：
- 责任上限：[X倍年费/合同总金额]
- 适用法律：[省份/司法管辖区]
- 数据处理：[需要DPA/需符合个保法/等]
- 禁止条款：[单方修改、永久许可授予等]

合同文本：
[在此粘贴合同文本]
```

**提示词2 — 特定条款深度分析**
```
请审查以下来自[合同类型]协议的赔偿条款，我方身份为[当事方角色]。请说明：
1. 赔偿义务的范围（哪些事件触发赔偿）
2. 赔偿是否为双向或单向
3. 任何例外或排除情形，以及是否可接受
4. 该条款是否与第[X]条的责任上限相冲突
5. 建议修订以符合[行业]合同的市场惯例

条款文本：
[在此粘贴条款]
```

**提示词3 — 政策合规核查**
```
请将以下合同与我方标准合同手册进行比对。标记每一个偏离我方政策立场的条款。对每处偏差，请说明：
- 章节引用
- 当前合同语言（摘要）
- 我方手册标准
- 差距严重程度（严重/高/中/低）
- 建议行动：接受/谈判/拒绝

我方手册：
[粘贴手册或主要立场]

合同：
[在此粘贴合同文本]
```

**提示词4 — 管理层风险摘要**
```
根据以下合同分析，为我方[CFO/CEO/采购副总裁]撰写一页纸的执行摘要，要求：
- 用通俗商业语言（无法律术语）列出五大风险事项
- 对每项风险，在可量化时说明财务敞口
- 建议哪些事项为交易障碍，哪些可谈判
- 建议谈判优先级顺序
- 给出总体风险评级：低/中/高/严重

分析内容：
[粘贴此前COCO分析或合同文本]
```

**提示词5 — 自动续约和终止陷阱扫描**
```
请审查以下合同，识别所有涉及以下内容的条款：
1. 自动续约条款（通知期限、退出窗口、费用递增）
2. 便利终止权（及任何限制条件）
3. 违约终止定义（以及是否对双方平衡）
4. 存续条款（终止后哪些义务继续有效）
5. 任何可能使我方陷入意外续约的通知要求

对每项发现，请说明章节、风险及建议的日历提醒或合同修正。

合同：
[在此粘贴合同文本]
```
